Checksum Examples.

The subdirectory contain 3 examples to show various features of the checksum calculation.

For simplicity they are only adapted to run in the simulator.

1. Simple Example
  - As simple as possible.
2. Single Example
  - As many features as possible. various checksums, Slow & big, but many examples
  - in this example also the prm file syntax is explained.
3. Separate Loader Example
  - in this example a loader checks the checksum of a second application and calls it if the checksum is OK.
    This is a advanced setup as multiple applications have to play together.
    It also contains some interrupt vector redirections written in assembly.
