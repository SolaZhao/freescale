Compiler MetrowerksHC12XCC
CodeWarrior HC12X C Compiler
Processor Expert CodeWarrior Plug-in:

Settings of the compiler are available in the "Compiler" folder in the CodeWarrior IDE "Target setting" window. The additional Processor Expert settings of the compiler you can find on page "Build options" in the CPU Component Inspector.

The linker settings are available in the "Linker" folder in the CodeWarrior IDE "Target setting" window.

Build options of Processor Expert

Settings of the Build options you can find on page "Build Options" in the CPU Component Inspector.

Default settings of the Build options are suitable for most applications. The important settings which can be change by users are the Stack size and the Memory model of the CPU. These settings is used for generating of the PRM file and should be change according to application requirements.
Please refer to page Settings for description of the Build options settings.
