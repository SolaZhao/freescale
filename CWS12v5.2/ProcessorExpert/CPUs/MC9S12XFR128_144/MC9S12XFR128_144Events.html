CPU Component MC9S12XFR128_144
Freescale HC9S12X family: MC9S12XFR128 in 144 pin package
Component Level: High
Category: CPU-Freescale-HCS12X-F
Events:
(Events are call-back functions called when an important event occurs. Please see the Embedded Components page for more information.)
  • EventModule - Name of the user module (without extension), where the events from this component are placed.
  • OnReset - This software event is called after reset.

      ANSIC prototype: void OnReset(void)

  • OnClockMonitorReset - This event is called immediately after fetching of the Clock monitor reset vector, i.e. Processor Expert initialization and startup initialization have not been done before (see generated code). This event allocates Clock monitor reset vector. This event is enabled only if the property Clock monitor is set to 'Force reset' value.

      ANSIC prototype: void OnClockMonitorReset(void)

  • OnIllegalOpcode - This event is called after illegal opcode is detected. This event allocates Illegal opcode interrupt vector.

      ANSIC prototype: void OnIllegalOpcode(void)

  • OnMMCRAMAccessViolation - This software event is called after MMC RAM access violation interrupt. This event allocates MMC RAM access violation interrupt vector.

      ANSIC prototype: void OnMMCRAMAccessViolation(void)

  • OnSwINT - This software event is called after software interrupt. This event allocates SWI interrupt vector.

      ANSIC prototype: void OnSwINT(void)

  • OnLvdStatusChanged - This event is triggered by any change of the LVD status bit. This event allocates LVI interrupt vector.

      ANSIC prototype: void OnLvdStatusChanged(void)

  • OnPllLockStatusChanged - This event is triggered by any change of the LOCK status bit. This event allocates PLL interrupt vector. This event is enabled only if PLL clock is enabled.

      ANSIC prototype: void OnPllLockStatusChanged(void)

  • OnSCMChanged - This event is triggered by any change of the SCM(Self clock mode) status bit of the Clock monitor module. This event allocates SCM interrupt vector. This event is enabled only if the property Clock monitor is set to 'Enter Self clock mode' value.

      ANSIC prototype: void OnSCMChanged(void)

  • OnSpuriousInterrupt - This event is called after spurious interrupt has been detected. This event allocates Spurious interrupt vector.

      ANSIC prototype: void OnSpuriousInterrupt(void)

For each generated event it is possible to set:
  • Procedure Name - name of procedure (written by user) that handles the event.
