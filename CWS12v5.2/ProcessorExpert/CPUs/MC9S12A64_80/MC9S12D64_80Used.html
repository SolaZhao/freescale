Freescale HC9S12 family: MC9S12A64 in 80 pinout

Here you can select peripherals that are used by an external module and blocked for using by Processor Expert™. It is guaranteed that Processor Expert™ will not use those peripherals and change their state/setting.
This setting is not supported in the Graphical IDE.
