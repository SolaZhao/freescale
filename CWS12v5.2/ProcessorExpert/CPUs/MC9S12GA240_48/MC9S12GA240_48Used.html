CPU Component MC9S12GA240_48
Freescale HC9S12 family: MC9S12GA240 in 48 pin package
This tab contains a list of all peripherals managed by the Processor Expert knowledge base. In some cases it is necessary to exclude certain peripherals in order to control them in an external user module or library. This page allows selecting which peripherals will be used by Processor Expert and which will not be used. When selected, it is guaranteed that Processor Expert will not use those peripherals or change their settings/states in any Component.

  • I/O Pins - list of pins
  • I/O Ports - list of ports
  • Timer and Watchdogs - list of timers and watchdogs
  • A/D and Serials channels - list of A/D converters and serial channels
