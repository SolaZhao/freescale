Component Init_ADC for HCS12X
Analog-to-Digital Converter (ADC)
Device Initialization
(Description of the component parameters.)
  • Device - Selection of the ATD device module.
  • Settings - Common ATD module settings.
    • Clock settings - Clock/timing settings of the module.
      • Settings only if ADC support optional discharge feature.
        • Discharge before sampling - If this property is set to yes, the internal sample capacitor is discharged before sampling the channel. This adds 2 A/D conversion clock periods to the sampling time.
          This item modifies the ATDCTL1[SMP_DIS] bit.
      • Sample time - This item selects the length of the second phase of the sample time in units of ATD conversion clock cycles.
        This item modifies the SMP0, SMP1 and eventually SMP2 (if exist) bits in the ATDCTL4 register.
      • Prescaler - This item defines the binary value of the ATD prescaler. The prescaling factor is calculated as follows: Prescaling factor = 2*(Prescaler+1).
        This item modifies the PRS[4:0] bits in the ATDCTL4 register.
      • Frequency - Calculated ATD module clock frequency (for information only). The value is calculated using the following formula: ATDclock = BusClock/(2*(Prescaler+1)) This item is influenced by the following properties: Prescaler.
      • Conversion time - Calculated time of one ATD conversion period (for information only.). This value is calculated using the following formula: Conversion time = (Discharge + ATD_Resolution + SampleTime + ATD_Resolution_Constant)/ATDClock;
        where ATD_Resolution_Constant has assigned this values:
        • for 8bit and 10 bit resolution ATD_Resolution_Constant = 5
        • for 12bit resolution is ATD_Resolution_Constant = 4
        and Discharged has assigned value depended on the ADC type:
        • on the S12ATD10B module Discharge = 2 A/D conversion clocks
        • on the ADC12B is the Discharge value controlled by the Discharge before sampling property.
        This item is influenced by the following properties: Prescaler, Sample Time, Result data formats and on the ADC12B is the Discharge controlled by the Discharge before sampling.
    • Conv. sequence length - This item controls the number of conversions per sequence.
      This item modifies the S1C, S2C, S4C and S8C bits in the ATDCTL3 register.
    • Conversion mode - This item selects whether conversion sequences are performed continuously or only once.
      This item modifies the SCAN bit in the ATDCTL5 register.
    • Channel sample mode - This property defines if the ATD converter samples only from the specified analog input channel or across channels for an entire conversion sequence.
      This item modifies the MULT bit in the ATDCTL5 register.
    • Result data formats - This property determines the resolution of the conversion and data format in which the conversion results are stored in the data registers.
      Derivatives with S12ATD10B module: This item modifies the DJM and DSGN bits in the ATDCTL5 register and the SRES8 bit in the ATDCTL4 register.
      Derivatives with ADC12B module: This item modifies the DJM bit in the ATDCTL3 register and SRES[1:0] bits in the ATDCTL1 register.
    • Fast flag clear all - This property determines the type of clearing of interrupt flags.
      Value "no" means normal ATD flag clearing.
      Value "yes" changes all ATD conversion complete flags to a fast clear sequence. Any access to a result register will cause the associate CCF flag to clear automatically.
      This item modifies the AFFC bit in the ATDCTL2 register.
    • Settings supported for Freescale HCS12 G, HY, P, VR, NE, B128, E256, HN64, HZ256, HZ64, KC128, KG128, KG256, KL128, KT256 and HCS12X derivatives only.
      • Wrap around channel - This item determines the channel for wrap around when doing multi-channel conversions.
        This item modifies the WRAP0, WRAP1, WRAP2 and WRAP3 bits in the ATDCTL0 register.
    • External trigger - This bit enables the external trigger on ATD channel or ETRIG pins. The possibilities of external trigger depends on ADC module version.
      This item modifies the ETRIGE bit in the ATDCTL2 register.
    • External trigger control - This item controls the sensitivity and the polarity of the external trigger signal.
      This item modifies ETRIGLE bit and ETRIGP bit in the ATDCTL2 register.
    • Conv. results are placed in - This property determines how the conversion results are placed in the data registers.
      This item modifies the FIFO bit in the ATDCTL3 register.

      There are 2 options:
      • Consecutive result registers: The conversion counter is not reset at the beginning or ending of a conversion sequence; conversion results are placed in consecutive result registers between sequences. The result register counter wraps around when it reaches the end of the result register file. The conversion counter value in ATDSTAT0 can be used to determine where in the result register file, the current conversion result will be placed.
      • Corresponding result register: The A/D conversion results map into the result registers based on the conversion sequence; the result of the first conversion appears in the first result register, the second result in the second result register, and so on.

    • Background debug freeze - This property allows to select the behavior of the ATD module during debugging when a breakpoint (Freeze Mode) is encountered.
      This item modifies the FRZ0 and FRZ1 bits in the ATDCTL3 register.
    • Special channel conversion - This property enables special channel conversion.
      Derivatives with S12ATD10B module: This item modifies the SC bit in the ATDTEST1 register.
      Derivatives with ADC12B module: This item modifies the SC bit in the ATDTEST5 register.
      Note: This property switch between two lists of available channels (external/special channels) in the Initial channel select property.
    • Settings only if ADC support ATD power down optional feature.
      • Power down in Wait mode - This property determines behavior of the ATD module when entering Wait Mode. Disabling the ATD module in the Wait Mode reduces MCU power consumption. Because analog electronic is turned off when powered down, the ATD requires a recovery time period after exit from Wait mode.
        This item modifies the AWAI bit in the ATDCTL2 register.
    • Settings only if ADC support Internal Clock in Stop Mode
      • AD conversions in STOP mode - When going into stop mode and this item is set to enabled the ATD conversion clock is automatically switched to the internally generated clock ICLK.
        Current conversion sequence will seamless continue. Conversion speed will change from prescaled bus
        frequency to the ICLK frequency (see ATD Electrical Characteristics in device description). The prescaler bits
        PRS4-0 in ATDCTL4 have no effect on the ICLK frequency.
        ATD Stop Recovery time is required to switch back to busclock based ATDCLK when leaving StopMode.
        This item modifies ICLKSTP bit in ATDCTL2 register.
    • Settings only if ADC support Compare function of ADC channels.
      • Number of automatic comparison - Compare settings: Number of automatic comparison in the sequence.
        This item does not modify any bit, it's used for optimal visualization of the compare settings only.
        One Item of the list looks like:
        Enable compare for conversion number - Set which conversion has to be compared with appropriate Compare value.
        This item modifies the ATDCMPE[n] bit, where n is the number of conversion in the sequence.
        There are 16 modes:
        • 0 - The following items are displayed in this mode:
          • Compare value - Value for compare with analog input, this item modifies the ATDDRn register, where n is the number of conversion in the sequence.
          • Set flag if result of conversion is - This item selects the operator for comparison of conversion results.
            This item modifies the ATDCMPHT[n] bit, where n is the number of conversion in the sequence.
        • 1 - See 0 mode for items available in this mode.
        • 2 - See 0 mode for items available in this mode.
        • 3 - See 0 mode for items available in this mode.
        • 4 - See 0 mode for items available in this mode.
        • 5 - See 0 mode for items available in this mode.
        • 6 - See 0 mode for items available in this mode.
        • 7 - See 0 mode for items available in this mode.
        • 8 - See 0 mode for items available in this mode.
        • 9 - See 0 mode for items available in this mode.
        • 10 - See 0 mode for items available in this mode.
        • 11 - See 0 mode for items available in this mode.
        • 12 - See 0 mode for items available in this mode.
        • 13 - See 0 mode for items available in this mode.
        • 14 - See 0 mode for items available in this mode.
        • 15 - See 0 mode for items available in this mode.
  • Pins/Signals - Module pins/signals settings.
    • External trigger - This property allows to select external trigger source.
      The following items are available only if the group is enabled (the value is "yes"):
      • External trigger source - Select external trigger source.
        This item modifies the ETRIGSEL bit and ETRIGCH[3:0] bits in the ATDCTL1 register.
      • Digital Input - This item controls the digital input buffer from the analog input pin.
        This item modifies the IENx bit in the ATDDIEN register.
        For proper functionality of ADC trigger:
        1) this property should be set to disabled if there is ETRIG pin/signal used
        2) this property should be set to enabled if there is ADC channel pin/signal used as trigger
    • ADC input pins - This item increases or decreases number of used input pins
      One Item of the list looks like:
      Input Pin0 - Input channel group
      • Pin - Select pin name
      • Digital Input - This item controls the digital input buffer from the analog input pin.
        This item modifies the IENx bit in the ATDDIEN register.
        For proper functionality of ADC channel input this property should be set to disabled.
      • Settings only if ADC channel is High voltage input type.
        • High voltage input - High voltage input (HVI)
          • Direct connection - Switch between the input voltage divider and direct connection on the pin routed to ADC.
            This item modifies the PTADIRp bit in register PTAp, where p is Port name, e.g.: PADIRL in PTAL for port L.
          • Impedance converter - Allow to bypass and power down impedance converter stage in the signal path from pin to ADC.
            This property takes effect only if the Direct connection is enabled.
            This item modifies the PTABYPp bit in register PTAp, where p is Port name, e.g.: PTABYPL in PTAL for port L.

            There are 2 options:
            • Active: Impedance converter stage is used in the signal path form pin to ADC.
            • Bypassed: Bypass and power down impedance converter stage in the signal path from pin to ADC.

          • Open input detection - Enables/disables open input detection mode.
            This property takes effect only if the Direct connection is disabled and MCU is not in STOP mode. This item modifies the PTTEp bit in register PTAp, where p is Port name, e.g.: PTTEL in PTAL for port L.
          • Pull Select - Selects pull device for open input detection.
            By default the pull down is selected as part of the input voltage divider. Only in open input detection mode the pull up could be selected.
            This property takes effect only if MCU is not in STOP mode. This item modifies the PTPSp bit in register PTAp, where p is Port name, e.g.: PTPSL in PTAL for port L.

            There are 2 options:
            • Pull down: Pull down select in analog mode for open input detection. Default state.
            • Pull up: Pull up select in analog mode for open input detection.

          • Select input ratio - Selects the input ratio of the HVI.

            There are 2 options:
            • Ratio L: Ratio L is selected.
            • Ratio H: Ratio H is selected.

  • Interrupts - Interrupt settings of the module.
    • Sequence Complete - Sequence Complete interrupt settings.
      • Sequence complete interrupt - Enable Sequence Complete interrupt.
        This item modifies the ASCIE bit in the ATDCTL2 register.
      • Interrupt - Interrupt vector (for information only).
      • Priority - Priority of sequence complete interrupt.
      • ISR name - Name of the interrupt service routine (ISR) invoked by this interrupt vector.
        Note: The routine must handle all interrupt flags correctly. The routine must be implemented in the user code, it is not generated by Processor Expert. Find external prototypes of the ISRs in the component module header file.
    • Settings only if ADC support Compare function of ADC channels.
      • Compare - Sequence Compare interrupt settings
        • Compare interrupt - Enable Compare interrupt.
          This item modifies the ACMPIE bit in the ATDCTL2 register.
        • Interrupt - Interrupt vector (for information only).
        • Priority - Priority of compare interrupt.
        • ISR name - Name of the interrupt service routine (ISR) invoked by this interrupt vector.
          Note: The routine must handle all interrupt flags correctly. The routine must be implemented in the user code, it is not generated by Processor Expert. Find external prototypes of the ISRs in the component module header file.
  • Initialization - Initialization options of the module.
    • Settings only if ADC support ATD power down optional feature.
      • Module - Module power up - this property provides on/off control over the AD module allowing reduced MCU power consumption.
        This item modifies the ADPU bit in the ATDCTL2 register.
    • Settings only if ADC has dedicated Reference voltage attenuator (RVA) device.
      • Reference voltage attenuator - Reference voltage attenuator (RVA)
        • Enable RVA - Enables/disables the reference voltage attenuator (RVA) module.
    • Start ADC conversion - This property determines if the AD conversion is started immediately after module initialization. The property Module must be enabled for this property to take effect.
      The following items are available only if the group is enabled (the value is "yes"):
      • Initial channel select - This item selects the initial input channel(s) to start the conversion with. In the case of multiple channel conversions (MULT=1), this selection represents the first channel to be examined in the conversion sequence.
        This item modifies CA, CB, CC and CD bits in the ATDCTL5 register.
        Note: If you don't see required channel please try to change the Special channel conversion property to switch channel list in Initial channel select property to view internal or special channels, like HVI or Voltage references.
    • Settings only if ADC support ATD power down optional feature.
      • Generate recovery delay - This property enables to place a delay loop in the init. code after the AD module is enabled. Because analog electronic is turned off when powered down, the ATD module requires a recovery time period (approx. 20us) after ADPU bit is enabled.
