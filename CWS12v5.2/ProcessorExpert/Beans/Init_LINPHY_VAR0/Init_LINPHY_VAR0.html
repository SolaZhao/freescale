Component Init_LINPHY for VAR0
LIN Physical Layer (LINPHY), currently supported for HCS12 family
Component Level: Peripheral Initialization
Category: CPU Internal Peripherals-Peripheral Initialization
This component provides initialization of the LIN Physical Layer (LINPHY), currently supported for HCS12 family module.

Table of the LINPHY interconnection options.

Signal routing Description How to do it in Processor Expert
SCI LINPHY interconnection setting:
SCIx is internally connected with LINPHY only.
On SCI side - Init_SCI or AsynchroSerial component select the signals:
RxD - LIN_RXD_to_SCIxRXD
TxD - LIN_TXD_to_SCIxTXD
On LINPHY side - Init_LINPHY component select the same signals:
Rx Output0 - LIN_RXD_to_SCIxRXD
Tx Input - LIN_TXD_to_SCIxTXD
TX LINPHY direct control setting:
Only Rx signal of SCIx is internally connected with LINPHY.
Tx signal is controled by SW in LINPHY data register.
On SCI side - Init_SCI or AsynchroSerial component select the signals:
RxD - LIN_RXD_to_SCIxRXD
TxD - Disabled
On LINPHY side - Init_LINPHY component select the signals:
Rx Output0 - LIN_RXD_to_SCIxRXD
Tx Input - LIN_SW_Control
Probe setting 1 (SCI LINPHY interconnection):
SCIx is internally connected with LINPHY.
Interface is accessible on two external probe pins.
On SCI side - Init_SCI or AsynchroSerial component select the signals:
RxD - LIN_RXD_to_SCIxRXD
TxD - LIN_TXD_to_SCIxTXD
On LINPHY side - Init_LINPHY component select the signals:
Rx Output0 - LIN_RXD_to_SCIxRXD
Tx Input - LIN_TXD_to_SCIxTXD
Tx Probe Output - Select external pin with LPDR1 feature
Rx Probe = Rx Outputx - Select external pin with LPRXD feature
Probe setting 2 (TX LINPHY direct control):
Only Rx signal of SCIx is internally connected with LINPHY.
Tx signal is controled by SW in LINPHY data register.
Interface is accessible on two external probe pins.
On SCI side - Init_SCI or AsynchroSerial component select the signals:
RxD - LIN_RXD_to_SCIxRXD
TxD - Disabled
On LINPHY side - Init_LINPHY component select the signals:
Rx Output0 - LIN_RXD_to_SCIxRXD
Tx Input - LIN_SW_Control
Tx Probe Output - Select external pin with LPDR1 feature
Rx Probe = Rx Outputx - Select external pin with LPRXD feature
Independent setting:
SCIx is disconnected from LINPHY.
Both peripheries have accessible his own external pins.
On SCI side - Init_SCI or AsynchroSerial component select the signals:
RxD - Select the one of external Rx pin
TxD - Select the one of external Tx pin
On LINPHY side - Init_LINPHY component select the signals:
Rx Outputx - Select external pin with LPRXD feature
Tx Input - Select external pin with LPTXD feature

Peripheral Initialization Components provide a low-level hardware approach to initialize registers of the peripheral module. They are intended for experienced users.
