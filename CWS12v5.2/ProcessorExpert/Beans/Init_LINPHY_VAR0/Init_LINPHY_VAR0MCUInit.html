Component Init_LINPHY for VAR0
LIN Physical Layer (LINPHY), currently supported for HCS12 family
Device Initialization
(Description of the component parameters.)
  • Device - Selection of the LINPHY device module.
  • Settings - Common LINPHY module settings
    • Slew rate - LINPHY slew rate selects.

      There are 3 options:
      • Normal: Normal slew rate (optimized for 20kBit/s).
      • Slow: Slow slew rate (optimized for 10kBit/s).
      • Fast: Fast slew rate (up to 250kBit/s).

    • Rx only mode - Enables/disables receive only mode.

      There are 2 options:
      • Enabled: Enables the receive only mode.
      • Disabled: Disables the receive only mode.

    • Wake up - Enables/disables wake up feature.

      There are 2 options:
      • Enabled: Enables the wake up feature.
      • Disabled: Disables the wake up feature.

    • Pull up - Selects the pull up impedance.

      There are 2 options:
      • Strong (30k): Selects the strong pull up with impedance = 30kOhm.
      • Weak (330k): Selects the weak pull up with impedance = 330kOhm.

  • Pins/Signals - Module pins/signals settings
    • LIN Bus - LIN Bus settings.
      • Pin - LIN bus pin.
    • LIN GND - LIN GND settings.
      • Pin - LIN GND pin.
    • Tx Input - LIN Tx input sources routing and allocation.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Pin - LIN Tx input source pin.
      • Tx Probe Output - Enables/disables additional Tx input signal routing to pin.
        The following items are available only if the group is enabled (the value is "Enabled"):
        • Pin - Tx probe output pin/signal.
    • Rx Output - LIN Rx output consumers routing and allocation.
      • Output 0 - Selects if LIN Rx consumer 0 is allocated and routed to LIN.
        The following items are available only if the group is enabled (the value is "Enabled"):
        • Pin 0 - LIN Rx output consumer pin/signal 0.
      • Output 1 - Selects if LIN Rx consumer 1 is allocated and routed to LIN.
        The following items are available only if the group is enabled (the value is "Enabled"):
        • Pin 1 - LIN Rx output consumer pin/signal 1.
      • Output 2 - Selects if LIN Rx consumer 2 is allocated and routed to LIN.
        The following items are available only if the group is enabled (the value is "Enabled"):
        • Pin 2 - LIN Rx output consumer pin/signal 2.
  • Interrupts - Interrupt settings of the module.
    • Over-current interrupt - Over-current interrupt settings.
      • Interrupt - Interrupt vector (for information only).
      • Interrupt priority
        • Interrupt priority - Interrupt priority
      • ISR Name - Name of the interrupt service routine (ISR) invoked by this interrupt vector.
        Note: The routine must handle all interrupt flags correctly. The routine must be implemented in the user code, it is not generated by Processor Expert. Find external prototypes of the ISRs in the component module header file.
      • Over-current interrupt - Enable/Disable Over-current interrupt.

        There are 2 options:
        • Enabled: Enabled invocation of the interrupt when Over-current condition occurred.
        • Disabled: Disabled invocation of the interrupt.

  • Initialization - Initialization options of the module.
    • Enable LINPHY - Enables/disables the LINPHY module.

      There are 2 options:
      • yes: Enables the LINPHY module.
      • no: Disables the LINPHY module.

    • Init state of Tx SW control - Initialize the LPDR1 bit - Tx data SW control bit. This setting takes effect if the LIN_SW_Control is select as LIN Tx source.

      There are 2 options:
      • 0: Set the LPDR1 bit to 0.
      • 1: Set the LPDR1 bit to 1.
