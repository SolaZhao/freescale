Component Init_DAC for HCS12
Digital to Analog Converter (DAC)
Device Initialization
(Description of the component parameters.)
  • Device - Selection of the DAC device module.
  • Settings - Common DAC module settings.
    • Data Justification - This property controls the justification of the data in the DAC data register (DACD). If set to left-justified, the data to be converted must be written to left justified DACD and the right justified DACD register will read zeroes. If set to right-justified, the data to be converted is written to right justified DACD register and left justified DACD register reads zeroes.
      This item modifies the DJM bit in the DACC0 register.

      There are 2 options:
      • Left-justified: Left justified data in DAC data register
      • Right-justified: Right justified data in DAC data register

    • Data Representation - This property selects between signed and unsigned conversion data representation in the DAC data register. Signed data is represented as 2’s complement.
      This item modifies the DSNG bit in the DACC0 register.

      There are 2 options:
      • Unsigned: Unsigned data representation in DAC data register
      • Signed: Signed data representation in DAC data register

    • DAC Stop in WAIT mode - This property selects the behaviour of the DAC module during WAIT mode.
      This item modifies the DACWAI bit in the DACC0 register.

      There are 2 options:
      • Enabled: DAC is enabled during WAIT mode
      • Disabled: DAC is disabled and powered down during WAIT mode

  • Pins - Module pins/signals settings.
    • Output pin - This property enables/disables the output pin of the DAC module.
      This item modifies the DACOE bit in the DACC0 register.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Pin - Selection of the DAC output pin.
  • Initialization - Initialization options of the module
    • Enable module - This item enables or disables the module.
      This item modifies the DACE bit in the DACC0 register.
    • Data Register value - Value of this item will be used to preset either left justified or right justified DACD register of the DAC module. Which DACD register will be used depends on the selected Data Justification mode.
      The admissible range of this property depends on the selected Data Representation mode.
      This item modifies the DACD register.
