Component FreeCntr
Time measurement based on counter value (suitable for short time periods)
Component Level: Low
Category: CPU Internal Peripherals-Timer
Component types:
(User types are derived from basic types and they are designed for usage in the driver interface. They are declared in the generated code.)
  • ComponentName_TTimerValue = dlong - Counter or compare value, width depends on timer width, 8, 16, 32 or 64 bits.
Component constants:
(Constants defined by this component for usage in the driver interface. They are defined in the generated code.)
  • ComponentName_PERIOD_VALUE - Initial period/offset value in ticks of the timer (see property Period/offset). It is available only if the component is enabled in high speed mode.
  • ComponentName_PERIOD_VALUE_HIGH - Period value in ticks of the timer in high speed mode (see property Period/offset). It is available only if the component is enabled in high speed mode.
  • ComponentName_PERIOD_VALUE_LOW - Period value in ticks of the timer in low speed mode (see property Period/offset). It is available only if the component is enabled in low speed mode.
  • ComponentName_PERIOD_VALUE_SLOW - Period value in ticks of the timer in slow speed mode (see property Period/offset). It is available only if the component is enabled in slow speed mode.
  • ComponentName_PRESCALER_VALUE - Initial prescaler value of the timer. It is available only if the component is enabled in high speed mode.
  • ComponentName_PRESCALER_VALUE_HIGH - Prescaler value of the timer in high speed mode. It is available only if the component is enabled in high speed mode.
  • ComponentName_PRESCALER_VALUE_LOW - Prescaler value of the timer in low speed mode. It is available only if the component is enabled in low speed mode.
  • ComponentName_PRESCALER_VALUE_SLOW - Prescaler value of the timer in slow speed mode. It is available only if the component is enabled in low speed mode.
  • ComponentName_COUNTER_INPUT_CLOCK_HZ_HIGH - Counter input clock frequency in high speed mode [Hz] (integer number). It is available only if the component is enabled in high speed mode.
  • ComponentName_COUNTER_INPUT_CLOCK_HZ_LOW - Counter input clock frequency in low speed mode [Hz] (integer number). It is available only if the component is enabled in low speed mode.
  • ComponentName_COUNTER_INPUT_CLOCK_HZ_SLOW - Counter input clock frequency in slow speed mode [Hz] (integer number). It is available only if the component is enabled in low speed mode.
  • ComponentName_COUNTER_INPUT_CLOCK_HZ - Initial counter input clock frequency [Hz] (integer number). It is available only if the component is enabled in high speed mode.
  • ComponentName_UP_COUNTER - This constant is defined if the counter counts upwards, e.g. from 0 to maximal value.
  • ComponentName_DOWN_COUNTER - This constant is defined if the counter counts down, e.g. from maximal value to 0.
  • ComponentName_RESET_ON_COMPARE - This constant is defined if Reset counter on compare property is selected.
