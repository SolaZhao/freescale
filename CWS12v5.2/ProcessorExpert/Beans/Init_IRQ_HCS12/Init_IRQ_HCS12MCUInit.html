Component Init_IRQ for HCS12
External Interrupt Module (IRQ)
Device Initialization
(Description of the component parameters.)
  • Device - Selection of the device module.
  • Pins - Module pins/signals settings.
    One Item of the list looks like:
    Pin1 - Group of external interrupt pin. Item enables settings and allocation of the external interrupt pin.
    • Pin - External interrupt pin
    • Pull resistor - Setting of the pull resistor of the external interrupt pin. This item modifies the appropriate 'Pull-up port Enable' bit in the pull control register.
      In case selected pin has polarity feature and pull-up or pull-down value is selected, this item modifies corresponding polarity register.
    • Triggering sensitivity - Select triggering sensitivity of the external interrupt pin.
      IRQ pin: This item modifies the IRQE bit in the IRQCR register.
      XIRQ pin: The edge cannot be selected.

      There are 6 options:
      • rising or falling edge: One of the following edge type will be selected: rising or falling
      • rising edge: Rising edge required
      • falling edge: Falling edge required
      • both edges: Both edges (rising and falling) required
      • high level: High level of the signal
      • low level: Low level of the signal

    • Interrupt - IRQ interrupt settings
      • Interrupt vector - Interrupt vector (for information only)
      • Interrupt - Enable/disable the interrupt.
        IRQ pin: This item modifies the IRQEN bit in the IRQCR register.
        XIRQ pin: This item modifies the X bit in the condition code register (CCR) and XIRQEN bit in IRQCR if the bit is present.
      • Interrupt priority - Priority of the interrupt.
      • ISR name - Name of the interrupt service routine (ISR) invoked by this interrupt vector.
        The user code of the routine should handle all respective interrupt flags correctly.
