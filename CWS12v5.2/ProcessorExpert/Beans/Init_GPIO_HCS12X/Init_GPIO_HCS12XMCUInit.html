Component Init_GPIO for HCS12X
General Purpose Input Output (GPIO)
Device Initialization
(Description of the component parameters.)
  • Device - Selection of the GPIO device module.
  • Settings - Common GPIO module settings.
    • Port control - Select if the pins of the device are controlled individually or as the entire I/O port. If the entire I/O port is selected only common settings for GPIO function are available and all pins of the port have same settings.
      The following items are available only if the group is enabled (the value is "Entire I/O port"):
      • Direction - Set direction of the entire I/O port.
        This item modifies the respective port direction data register.
      • Output value - Setting of the initial output value of the entire I/O port. If the port direction is set to input the value is stored in data latch of the device and it automatically appears on the port when the direction changed to output.
        Either value from the range or "no initialization" option can be selected, default setting is "no initialization".
        This item modifies the respective port data register.
      • Pull resistor - Setting of the pull resistor of the entire I/O port.
        This item modifies the respective bits in the respective pull-up enable register and eventually in the polarity select register.

        There are 4 options:
        • no initialization: No initialization of the pull up resistor or setting not available.
        • no pull resistor: No internal pull up resistor present.
        • pull up: Internal pull up resistor present.
        • pull down: Internal pull down resistor present.

      • Open drain - Setting of the open drain option of the entire I/O port.
        This item modifies the respective bits in the port wired-or mode register.
      • Setting is available only if the module requires to enable digital input buffer from the analog input pins.
        • Digital input - Setting of the digital input buffer control for the analog input pins of the entire I/O port.
          This item modifies the respective bits in the port digital input enable register.
      • Setting is available only if the module enables interrupt polarity setting but no pull up selection.
        • Interrupt polarity - Setting of the interrupt polarity of the entire I/O port.
          This item modifies the respective bits in the port polarity select register.

          There are 2 options:
          • Rising: Rising edge selected
          • Falling: Falling edge selected

      • Setting is available only if the module has over current detect feature.
        • Over current detect - Setting of the iover current detect feature of the entire I/O port.
          This item modifies the respective bits in the port over current setting register.
  • Pins - Module pins/signals settings (for information only). Setting is enabled if Port control property set to Individual pins and disabled if port control set to the Entire I/O port.
    The following items are available only if the group is enabled (the value is "Enabled"):
    • Pin0 - This description is same for all pins of GPIO port.
      Group of pin x for GPIO. Item enables settings and allocation of the GPIO pin x.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Pin - Pin x of the GPIO port (for information only).
      • Direction - Set pin x direction to input or output.
        This item modifies the respective data direction register.
      • Output value - Setting of the initial output value = 0 or 1. If the pin x direction is set to input the value is stored in data latch of the device and it automatically appears on the pin x when the direction changed to output.
        Either value from the range or "no initialization" option can be selected, default setting is "no initialization".
        This item modifies the respective port data register.
      • Pull resistor - Setting of the pull resistor for the pin x.
        This item modifies the respective bit in the port input pull-up enable register and eventually in the polarity select register.

        There are 4 options:
        • no initialization: No initialization of the pull up resistor or setting not available.
        • no pull resistor: No internal pull up resistor present.
        • pull up: Internal pull up resistor present.
        • pull down: Internal pull down resistor present.

      • Open drain - Setting of the open drain option for the pin x.
        This item modifies the respective bit in the port wired-or mode register.
      • Setting is available only if the pin requires to enable digital input buffer from the analog input pin.
        • Digital input - Setting of the digital input buffer control for the analog input pin x.
          This item modifies the respective bit in the port digital input enable register.
      • Setting is available only if the pin enables interrupt polarity setting but no pull up selection.
        • Interrupt polarity - Setting of the interrupt polarity for the pin x.
          This item modifies the respective bit in the port polarity select register.

          There are 2 options:
          • Rising: Rising edge selected
          • Falling: Falling edge selected

  • Settings are available only if module supports an interrupt.
    • Interrupts - Interrupt settings of the module.
      • Settings are available only if module supports Keypad wakeup interrupt.
        • Keypad Wakeup Interrupt - Keypad Wakeup Interrupt settings and allocation.
          • Interrupt - Interrupt vector used for the interrupt request from the port (for information only).
          • Pin 0 Interrupt - Enables or disables the edge sensitive external interrupt associated with the pin x.
            This item modifies the appropriate bit in the port interrupt enable register.
          • Pin 1 Interrupt - Enables or disables the edge sensitive external interrupt associated with the pin 1.
            This item modifies the appropriate bit in the port interrupt enable register.
          • Pin 2 Interrupt - Enables or disables the edge sensitive external interrupt associated with the pin 2.
            This item modifies the appropriate bit in the port interrupt enable register.
          • Pin 3 Interrupt - Enables or disables the edge sensitive external interrupt associated with the pin 3.
            This item modifies the appropriate bit in the port interrupt enable register.
          • Pin 4 Interrupt - Enables or disables the edge sensitive external interrupt associated with the pin 4.
            This item modifies the appropriate bit in the port interrupt enable register.
          • Pin 5 Interrupt - Enables or disables the edge sensitive external interrupt associated with the pin 5.
            This item modifies the appropriate bit in the port interrupt enable register.
          • Pin 6 Interrupt - Enables or disables the edge sensitive external interrupt associated with the pin 6.
            This item modifies the appropriate bit in the port interrupt enable register.
          • Pin 7 Interrupt - Enables or disables the edge sensitive external interrupt associated with the pin 7.
            This item modifies the appropriate bit in the port interrupt enable register.
          • Interrupt Priority - Priority of the Keypad Wakeup interrupt.
          • ISR name - Name of the interrupt service routine (ISR) invoked by this interrupt vector.
            The user code of the routine should handle all respective interrupt flags correctly. The routine must be implemented in the user code, it is not generated by Processor Expert. Find external prototypes of the ISRs in the component module header file.
      • Settings are available only if module supports Over current interrupt
        • Over current interrupt - Over current interrupt settings and allocation.
          • Interrupt - Interrupt vector used for the interrupt request from the port (for information only).
          • Settings are available only if pin 2 supports Over current interrupt
            • Pin 2 Interrupt - Enables or disables the over current interrupt associated with the pin 2.
              This item modifies the appropriate bit in the port over current interrupt enable register.
          • Interrupt Priority - Priority of the Keypad Wakeup interrupt.
          • ISR name - Name of the interrupt service routine (ISR) invoked by this interrupt vector.
            The user code of the routine should handle all respective interrupt flags correctly. The routine must be implemented in the user code, it is not generated by Processor Expert. Find external prototypes of the ISRs in the component module header file.
