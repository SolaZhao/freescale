Component Init_SSD for HCS12X
Stepper Stall Detector (SSD)
Device Initialization
(Description of the component parameters.)
  • Device - Selection of the SSD device module
  • Settings - Common SSD module settings
    • Clock settings - Clock/timing settings of the module.
      • Down counter prescaler - Bus clock prescaler for the modulus down counter frequency.
        This item modifies the PRE bit in the MDCCTL register.
      • Down counter frequency - Down counter frequency (for information only]. This item is influenced by the following properties: Down counter prescaler.
      • Accumulator sample prescaler - This field sets the accumulator sample frequency by pre-scaling the bus frequency by a selected factor.
        This item modifies the ACLKS[0:1] bits in the SSDCTL register.
      • Accumulator sample - Accumulator sample frequency (for information only]. This item is influenced by the following properties: Accumulator sample prescaler.
      • Modulus down-counter - Value of the SSD modulus down-counter.
        This item modifies the MDCCNT[15:0] register.
      • Down-counter period - Period of the counter underflow (for information only). This item is influenced by the following properties: Modulus down counter, Down counter prescaler.
    • Down-counter enable - Enable/Disable modulus down-counter.
      This item modifies the MCEN bit in the MDCCTL register.
    • Read count register - Return value by reading modulus count register (MDCCNT).
      This item modifies the RDMCL bit in the MDCCTL register.
    • Modulus mode - If normal, then the modulus counter counts down from the value in the counter register and will stop at 0x00. If iterative and the counter reaches 0x00, the counter loaded with the latest value written to the modules counter register.
      This item modifies the MODMC bit in the MDCCTL register.
    • Drive coil - If the drive coil is enabled, this coil is driven. If is disabled, this coil is disconnected or drivers turned off.
      This item modifies the DCOIL bit in the RTZCTL register.
    • Integration - If value is integration, the coil is non-driven, and if is blanking, the coil is being recirculation.
      This item modifies the ITG bit in the RTZCTL register.
    • Polarity - This property determines witch end of the non-driven coil is routed to the sigma-delta converter during conversion or integration mode.
      This item modifies the POL bit in the RTZCTL register.
    • Sigma-Delta Converter - This property provides on/off control for the sigma-delta converter allowing reduced MCU power consumption.
      This item modifies the SDCPU bit in the SSDCTL register.
    • Recirculation in blanking mode - This property determines if the coil is recirculated via VDDM or via VSSM.
      This item modifies the RCIR bit in the RTZCTL register.
    • Stepper motor select - This property select one of two possible motors to be used for stall detection.
      This item modifies the SMS bit in the RTZCTL register.
    • Full step state - This field indicates one of the four possible full step state.
      This item modifies the STEP[1:0] bits in the RTZCTL register.
    • Stop in wait mode - Determines if the component is stopped in wait mode.
      This item modifies the SSDWAI bit in the SSDCTL register.
  • Pins - Module pins/signals settings.
    • Cosine pins - Cosine coil pins for motor
      • Minus pin - Cosine minus pin.
      • Minus pull resistor - Setting of the pull resistor for the cosine minus pin.
        This item modifies the respective bit in the port input pull-up enable register and eventually in the polarity select register.
      • Minus pin open drain - Setting of the open drain option for the cosine minus pin.
        This item modifies the respective bit in the port wired-or mode register.
      • Plus pin - Cosine plus pin.
      • Plus pull resistor - Setting of the pull resistor for the cosine plus pin.
        This item modifies the respective bit in the port input pull-up enable register and eventually in the polarity select register.
      • Plus pin open drain - Setting of the open drain option for the cosine plus pin.
        This item modifies the respective bit in the port wired-or mode register.
    • Sine pins - Sine coil pins for motor
      • Minus pin - Sine minus pin.
      • Minus pull resistor - Setting of the pull resistor for the sine minus pin.
        This item modifies the respective bit in the port input pull-up enable register and eventually in the polarity select register.
      • Minus pin open drain - Setting of the open drain option for the sine minus pin.
        This item modifies the respective bit in the port wired-or mode register.
      • Plus pin - Sine plus pin.
      • Plus pull resistor - Setting of the pull resistor for the sine plus pin.
        This item modifies the respective bit in the port input pull-up enable register and eventually in the polarity select register.
      • Plus pin open drain - Setting of the open drain option for the sine plus pin.
        This item modifies the respective bit in the port wired-or mode register.
  • Interrupts - Interrupts settings of the module.
    • SSD interrupt - SSD interrupt settings.
      • Interrupt - Interrupt vector (for information only).
      • Counter underflow interrupt - Enable/Disable SSD counter underflow interrupt.
        This item modifies MCZIE bit in the MDCCTL register.
      • Accumulator overflow interrupt - Enable/Disable SSD accumulator overflow interrupt.
        This item modifies AOVIE bit in the MDCCTL register.
      • Interrupt priority - Priority of SSD interrupt.
      • ISR name - Name of the interrupt service routine (ISR) invoked by this interrupt vector.
        Note: The routine must handle all interrupt flags correctly. The routine must be implemented in the user code, it is not generated by Processor Expert. Find external prototypes of the ISRs in the component module header file.
  • Initialization - Initialization options of the module.
    • Enable SSD system - Enable SSD system .
      This item modifies the RTZE bit in the SSDCTL register.
