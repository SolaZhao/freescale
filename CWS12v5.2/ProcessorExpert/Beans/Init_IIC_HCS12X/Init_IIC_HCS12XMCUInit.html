Component Init_IIC for HCS12X
Inter-IC Bus (IIC)
Device Initialization
(Description of the component parameters.)
  • Device - Selection of the IIC communication device module.
  • Settings - Common IIC module settings.
    • Clock settings - Clock/timing settings of the module.
      • Multiplier Factor - Multiplier Factor.
        This item modifies IBC[7:6] bits in the IBFD register.

        There are 3 options:
        • 1: For frequency enumeration
        • 2: For frequency enumeration
        • 4: For frequency enumeration

      • SCL Tap (Clocks) - SCL Tap (Clocks).
        This item modifies IBC[2:0] bits in the IBFD register.

        There are 8 options:
        • 5: The value is number of clocks
        • 6: The value is number of clocks
        • 7: The value is number of clocks
        • 8: The value is number of clocks
        • 9: The value is number of clocks
        • 10: The value is number of clocks
        • 12: The value is number of clocks
        • 15: The value is number of clocks

      • Tap to Tap (Clocks) - Tap to Tap (Clocks).
        This item modifies IBC[5:3] in the IBFD register.

        There are 8 options:
        • 1: The value is number of clocks
        • 2: The value is number of clocks
        • 4: The value is number of clocks
        • 8: The value is number of clocks
        • 16: The value is number of clocks
        • 32: The value is number of clocks
        • 64: The value is number of clocks
        • 128: The value is number of clocks

      • SCL frequency - SCL frequency. This item is influenced by the following properties: Multiplier Factor, SCL Tap (Clocks), Tap to Tap (Clocks). For information only.
      • SDA Hold - SDA hold. This item is influenced by the following properties: Multiplier Factor, SCL Tap (Clocks), Tap to Tap (Clocks). For information only.
      • SCL hold (Start) - SCL hold (Start). This item is influenced by the following properties: Multiplier Factor, SCL Tap (Clocks), Tap to Tap (Clocks). For information only.
      • SCL hold (Stop) - SCL hold (Stop). This item is influenced by the following properties: Multiplier Factor, SCL Tap (Clocks), Tap to Tap (Clocks). For information only.
    • Settings only if IIC supports 10-bit addressing and general call features.
      • General Call - The property enables general call address reception.
        This item modifies the GCEN bit in the IBCR2 register.
      • Address mode - The property controls number of bits used for the slave address.
        This item modifies ADTYPE bit in the IBCR2 register.
    • Trans/Receiv mode select - This item selects the direction of master and slave transfers.
      This item modifies the Tx/Rx bit in the IBCR register.

      There are 2 options:
      • Receive: Receive mode is selected
      • Transmit: Transmit mode is selected

    • Transmit Acknowledge - This item specifies the value driven onto SDA during data acknowledge cycles for both master and slave receivers.
      This item modifies the TXAK bit in the IBCR register.
    • Slave Address - This item specifies an address, the IIC Bus will respond to when addressed as a slave.
      This item modifies ADR[7:1] bits in the IBAD register and if 10-bit addressing is used this item also modifies ADR[10:8] bits in the IBCR2 register.
    • Stop in WAIT mode - I-Bus Interface Stop in WAIT mode.
      This item modifies the IBSWAI bit in the IBCR register.
  • Pins - Module pins/signals settings.
    • SDA pin - Settings related to SDA pin of the IIC peripheral.
      • SDA pin - Bidirectional Serial Data line (SDA).
      • SDA pin pull resistor - Setting of the pull resistor for the SDA pin.
        This item modifies the respective bit in the port input pull-up enable register and eventually in the polarity select register.
      • SDA pin open drain - Setting of the open drain option for the SDA pin.
        This item modifies the respective bit in the port wired-or mode register.
    • SCL pin - Settings related to SCL pin of the IIC peripheral.
      • SCL pin - Bidirectional Serial Clock Line (SCL).
      • SCL pin pull resistor - Setting of the pull resistor for the SCL pin.
        This item modifies the respective bit in the port input pull-up enable register and eventually in the polarity select register.
      • SCL pin open drain - Setting of the open drain option for the SCL pin.
        This item modifies the respective bit in the port wired-or mode register.
  • Interrupts - Interrupts settings of the module.
    • IIC interrupt - IIC interrupt settings.
      • Interrupt - Interrupt vector (for information only).
      • IIC interrupt - Enable/Disable IIC interrupt.
        This item modifies IBIE bit in the IBCR register.
      • Interrupt priority - Priority of IIC interrupt.
      • ISR name - Name of the interrupt service routine (ISR) invoked by this interrupt vector.
        Note: The routine must handle all interrupt flags correctly. The routine must be implemented in the user code, it is not generated by Processor Expert. Find external prototypes of the ISRs in the component module header file.
  • Initialization - Initialization options of the module.
    • Bus Enable - This item enables/disables the module. If "no" is selected, the module is disabled and held in software reset state.
      This item modifies IBEN bit in the IBCR register.
