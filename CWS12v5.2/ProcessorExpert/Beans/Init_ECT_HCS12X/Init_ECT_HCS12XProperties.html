Component Init_ECT for HCS12X
Enhanced Capture Timer (ECT)
Component Level: Peripheral Initialization
Category: CPU Internal Peripherals-Peripheral Initialization
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • Device - Selection of the ECT module
  • Settings - Common ECT module settings.
    • Clock settings - Clock/timing settings of the module.
      • Prescaler - Timer Prescaler Select.
        This item modifies the PR0, PR1 and PR2 bits in the TSCR2 register.(Available only For HCS12X supporting Legacy mode.)

        There are 8 options:
        • Bus Clock / 1: Prescale Factor
        • Bus Clock / 2: Prescale Factor
        • Bus Clock / 4: Prescale Factor
        • Bus Clock / 8: Prescale Factor
        • Bus Clock / 16: Prescale Factor
        • Bus Clock / 32: Prescale Factor
        • Bus Clock / 64: Prescale Factor
        • Bus Clock / 128: Prescale Factor

      • Timer Clock Source - Timer Clock Source.
        This item modifies the CLK0 and CLK1 bits in the PACTL register.

        There are 4 options:
        • Timer prescaler: Use timer prescaler clock as timer counter clock
        • PACLK: Use PACLK as input to timer counter clock
        • PACLK/256: Use PACLK/256 as timer counter clock frequency.
        • PACLK/65536: Use PACLK/65536 as timer counter clock frequency.

      • Frequency - A input frequency of main ECT counter. This item is influenced by the following properties: Prescaler and Timer Clock Source..
      • Settings supported for Freescale HCS12X derivatives only.
        • Precision timer - Enables precision timer. All bits in the DLYCT register are used for the delay selection, all bits of the PTPSR register are used for Precision Timer Prescaler Selection, and all bits of PTMCPSR register are used for the prescaler Precision Timer Modulus Counter Prescaler selection. This item modifies the PRNT bit in the TSCR1 register.
          The following items are available only if the group is enabled (the value is "Enabled"):
          • Precision timer prescaler - This item specifies the division rate of the main Timer prescaler. This item modifies the PTPSR register.
          • Precision timer modulus counter prescaler - This item specifies the division rate of the modulus counter prescaler. This item modifies the PTMCPSR register.
    • Fast Flag Clear - Timer Fast Flag Clear All. This has the advantage of eliminating software overhead in a separate clear sequence.
      This item modifies the TFFCA bit in the TSCR1 register.
    • Timer Counter Reset - This item allows the timer counter to be reset by a successful output compare 7 event. This mode of operation is similar to an up-counting modulus counter.
      This item modifies the TCRE bit in the TSCR2 register.
    • Stop in wait mode - Disable the timer module when the MCU is in the wait mode. Timer interrupts cannot be used to get the MCU out of wait. Pulse accumulators and modulus down counter are also affected.
      This item modifies the TSWAI bit in the TSCR1 register.
    • Stop in freeze mode - Disable the timer and modulus counter whenever the MCU is in freeze mode. Pulse accumulators will not stop.
      This item modifies the TSFRZ bit in the TSCR1 register.
    • Delay Counter - If enabled, after detection of a valid edge on input capture pins, the delay counter counts the pre-selected number of bus clock cycles, then it will generate a pulse on its output. The pulse is generated only if the level of input signal, after the preset delay, is the opposite of the level before the transition.
      This property affects edge detectors on channel 0, 1, 2 and 3 (captures and pulse accumulators) This item modifies the [DLY0:DLY1] bits or [DLZ0:DLY7] bits (if Precision timer is supported and enabled) in the DLYCT register.
    • Mode - Input Control Latch or Queue Mode.
      Queue Mode - The main timer value is memorized in the IC register by a valid input pin transition. With a new occurrence of a capture, the value of the IC register will be transferred to its holding register (if enabled - see property Buffer) and the IC register memorizes the new timer value.
      Latch Mode - Latching function occurs when modulus down-counter reaches zero or can be invoke by SW. With a latching event the contents of IC registers and 8-bit pulse accumulators are transferred to their holding registers. 8-bit accumulators are cleared. This item modifies the LATQ bit in the ICSYS register.
    • Buffer - Enable Input Capture and pulse accumulator holding registers.
      In latch mode, the holding registers are always enabled. This item modifies the BUFFEN bit in the ICSYS register.
    • Timer flag mode - Setting "New value in latch" allows a timer interrupt (consequently user event OnCapture) to be generated after capturing two values in the capture and holding registers instead of generating an interrupt for every capture.
      This property is active only in Queue mode with enabled buffers (property Mode and property Buffer).
      This property affects interrupts related with channel 0, 1, 2 and 3. This item modifies the TFMOD bit in the ICSYS register.
    • 8-bit Pulse accumulator max. count - If this property is "yes", 8-bit pulse accumulators counters will be not incremented over value $FF (to value $00), while invoking interrupt whenever pulse appears on the input. The value $FF indicates a count of 255 or more. This item modifies the PACMX bit in the ICSYS register.
    • Share input action - Share input action
      • Share capture channels 3 & 7 - If this property is "yes" then channel input 3 causes the same action on the channel 7.
      • Share capture channels 2 & 6 - If this property is "yes" then channel input 2 causes the same action on the channel 6.
      • Share capture channels 1 & 5 - If this property is "yes" then channel input 1 causes the same action on the channel 5.
      • Share capture channels 0 & 4 - If this property is "yes" then channel input 0 causes the same action on the channel 4.
    • Accumulator A - This group covers all settings for accumulator A.
      • Accumulator counter - Accumulator A counter device selection.
      • Pulse Accumulator Mode - Pulse Accumulator Mode.
        This item modifies the PAMOD bit in the PACTL register.
        There are 2 modes:
        • Event counter - This bit is active only when the Pulse Accumulator A is enabled. The following items are displayed in this mode:
          • Edge Control - Pulse Accumulator Edge Control.
            This item modifies the PEDGE bit in the PACTL register.

            There are 2 options:
            • Rising: Rising edges on PT7 pin cause the count to be incremented.
            • Falling: Falling edges on PT7 pin cause the count to be incremented.

        • Gated time accumulation - This bit is active only when the Pulse Accumulator A is enabled. The following items are displayed in this mode:
          • Edge Control - Pulse Accumulator Edge Control.
            This item modifies the PEDGE bit in the PACTL register.

            There are 2 options:
            • Div. by 64 clock enabled with pin low level: PT7 input pin low enables bus clock divided by 64 to Pulse Accumulator and the trailing rising edge on PT7 sets the PAIF flag.
            • Div. by 64 clock enabled with pin high level: PT7 input pin high enables bus clock divided by 64 to Pulse Accumulator and the trailing falling edge on PT7 sets the PAIF flag

      • Count register 2 - Pulse Accumulators Count Register 2. The two 8-bit pulse accumulators PAC3 and PAC2 are cascaded to form the PACA 16-bit pulse accumulator.
        This item modifies all bits in PACN2 register.
      • Count register 3 - Pulse Accumulators Count Register 3. The two 8-bit pulse accumulators PAC3 and PAC2 are cascaded to form the PACA 16-bit pulse accumulator.
        This item modifies all bits in PACN3 register.
    • Accumulator B - This group covers all settings for accumulator B.
      • Accumulator counter - Accumulator B counter device selection.
      • Count register 0 - Pulse Accumulators Count Register 0. The two 8-bit pulse accumulators PAC1 and PAC0 are cascaded to form the PACB 16-bit pulse accumulator.
        This item modifies all bits in PACN0 register.
      • Count register 1 - Pulse Accumulators Count Register 1. The two 8-bit pulse accumulators PAC1 and PAC0 are cascaded to form the PACB 16-bit pulse accumulator.
        This item modifies all bits in PACN1 register.
    • Modulus Counter - Settings of modulus Counter
      • Modulus counter - Modulus counter device selection.
      • Modulus mode - If this property is "no", counter counts once from the value written to it and will stop at 0.
        In other case, when the counter reaches 0, the counter is loaded with the latest value written to the modulus count register.
        This item modifies the MODMC bit in the MCCTL register.
      • Read - Read Modulus Down-Counter Load.
        This item modifies the RDMCL bit in the MCCTL register.

        There are 2 options:
        • Value of the count register: Reads of the modulus count register will return the present value of the count register.
        • Content of the load register: Reads of the modulus count register will return the contents of the load register.

      • Count register - Modulus Down-Counter Count Register .A full access for the counter register should take place in one clock cycle.
        This item modifies all bits in MCCNT register.
      • Clock settings - Clock settings of modulus counter.
        • Prescaler - Modulus Counter Prescaler select.
          This item modifies the MCPR1 and MCPR0 bits in the MCCTL register.
        • Frequency - Frequency value . This item is influenced by the following properties: ModulusPrescaler.
  • Channels - Settings of the ECT channels.
    • Channel 0 - This group covers all settings for channel 0.
      • Channel device - Channel 0 device selection.
      • Mode - Input Capture or Output Compare Channel Configuration.
        This item modifies the IOS0 bit in the TIOS register .

        There are 2 options:
        • Capture: The corresponding channel acts as an input capture
        • Compare: The corresponding channel acts as an output compare.

      • Compare Mask - Setting the OC7M0 will set the corresponding port to be an output port when the corresponding TIOS0 bit is set to be an output compare.
        This item modifies the OC7M0 bit in the OC7M register .
      • Compare Data - A channel 0 output compare can cause bits in the output compare 0 data register to transfer to the timer port data register depending on the output compare 0 mask register.
        This item modifies the OC7D0 bit in the OC7D register .
      • Toggle On Overflow - It toggles output compare pin on overflow.
        This item modifies the TOV0 bit in the TTOV register.
      • Output action - This item specifies the output action to be taken as a result of a successful OC0 compare.
        This item modifies the OM0 and OL0 bit in the TCTL2 register.

        There are 4 options:
        • Disconected: Timer disconnected from output pin logic.
        • Toggle output line: Toggle OCx output line.
        • Clear line to zero: Clear OCx output line to zero.
        • Set line to one: Set OCx output line to one.

      • Settings supported for Freescale HCS 12X derivates containing the OCPD register.
        • Output Compare Pin - Enables/Disables the timer channel IO port.
          This item modifies the OCPD0 bit in the OCPD register.
      • Input control - This item configures the input capture edge detector circuits.
        This item modifies the EDG0A and EDG0B bits in the TCTL2 register.

        There are 4 options:
        • Capture disabled: This item also configure the 8 bit pulse accumulators PAC0 - 3.
        • Capture on rising edges only: This item also configure the 8 bit pulse accumulators PAC0 - 3.
        • Capture on falling edges only: This item also configure the 8 bit pulse accumulators PAC0 - 3.
        • Capture on any edge: This item also configure the 8 bit pulse accumulators PAC0 - 3.

      • Capture/Compare register - Timer Input Capture/Output Compare Register. Writes to this register have no meaning or effect during input capture mode.
        This item modifies all bits in TC0 register.
      • Capture register Overwrite - Capture register 0 Overwrite.
        This item modifies the NOVW0 bit in the ICOVW register.
      • Channel 0 interrupt - Channel 0 interrupt settings.
        • Channel interrupt - Input Capture/Output Compare 0 Interrupt Enable.
          This item modifies the C0I bit in the TIE register.
        • Interrupt - Interrupt vector (for information only).
        • Priority - Priority of the Input Capture/Output Compare 0 Interrupt.
        • ISR name - Name of the interrupt service routine (ISR) invoked by this interrupt vector.
          Note: The routine must handle all interrupt flags correctly. The routine must be implemented in the user code, it is not generated by Processor Expert. Find external prototypes of the ISRs in the component module header file.
      • Pin - Channel 0 pins/signals settings. This item enables initialization of the pin and if enabled, pin is used exclusively by this component.
        The following items are available only if the group is enabled (the value is "Enabled"):
        • Pin - IOC0 - Input capture and Output compare channel 0.
        • Pin signal - Channel 0 pin signal.
        • Pin pull resistor - Setting of the pull resistor for the pin of ECT channel 0.
          This item modifies the respective bit in the port input pull-up enable register and eventually in the polarity select register.
        • Pin open drain - Setting of the open drain option for the pin of ECT channel 0.
          This item modifies the respective bit in the port wired-or mode register.
