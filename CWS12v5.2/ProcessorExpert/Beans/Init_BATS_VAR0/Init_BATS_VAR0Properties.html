Component Init_BATS for VAR0
Supply Voltage Sensor (BATS)., currently supported for HCS12, HCS12Z families
Component Level: Peripheral Initialization
Category: CPU Internal Peripherals-Peripheral Initialization
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • Device - Selection of the BATS device module.
  • Settings - BATS module settings
    • ADC connection - This property selects supply pin, which will be connected through the resistor chain to ground, and connect ADC channel (BATS_Output) to divided-down voltage.
    • Comparator - Settings related to comparator (level sense).
      • Level sense - This property selects supply pin, which will be connected through the resistor chain to ground, and enables Voltage Level Sense features measuring BVLC and BVHC.
      • Voltage high select - This property selects the trigger level for the Voltage Level High Condition (BVHC). See reference manual for specific voltage values.

        There are 2 options:
        • V_HBI_1: High Voltage Warning 1
        • V_HBI_2: High Voltage Warning 2

      • Voltage low select - This property selects the trigger level for the Voltage Level Low Condition (BVLC). See reference manual for specific voltage values.

        There are 4 options:
        • V_LBI_1: Low Voltage Warning 1
        • V_LBI_2: Low Voltage Warning 2
        • V_LBI_3: Low Voltage Warning 3
        • V_LBI_4: Low Voltage Warning 4

  • Pins/Signals - Module pins/signals settings.
    • Settings supported only if the BATS module has Vsense pin.
      • Battery voltage sense - Setting of the Battery voltage sense pin.
        • Pin - Battery voltage sense pin
        • Pin signal - Signal name of Pin
    • Voltage supply - Setting of the Voltage supply pin.
      • Pin - Voltage supply pin
      • Pin signal - Signal name of Pin
  • Interrupts - Interrupts settings of the module
    • BATS interrupt - BATS interrupt.
      • Interrupt - Interrupt vector (for information only).
      • ISR Name - Name of the interrupt service routine (ISR) invoked by this interrupt vector.
        The user code of the routine should handle all respective interrupt flags correctly. The routine must be implemented in the user code, it is not generated by Processor Expert. Find external prototypes of the ISRs in the component module header file.
      • High Voltage Interrupt - Enables High Voltage Interrupt. The interrupt is generated when Voltage Sense High Condition Bit changes.
      • Low Voltage Interrupt - Enables Low Voltage Interrupt. The interrupt is generated when Voltage Sense Low Condition Bit changes.
  • Initialization - Initialization options of the module.
    • Call Init method - The Init method of the component may be called after a power-on or reset (in initialization code) by PE.
      yes - The Init method is called by PE.
      no - The Init method is not called by PE, it should be called in the user's code.
