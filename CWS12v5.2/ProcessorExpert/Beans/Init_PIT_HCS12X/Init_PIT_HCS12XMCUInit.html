Component Init_PIT for HCS12X
Periodic Interrupt Timer (PIT) or Enhanced Programmable Interrupt Timer (EPIT)
Device Initialization
(Description of the component parameters.)
  • PIT device - Selection of the PIT device module"
  • Settings - Common PIT module settings.
    • Clock settings - Clock/timing settings of the PIT module.
      • Micro Timer 0 load register - Modulus down counter value of the Micro Timer 0.
        This item modifies the PITMTLD0 register.
      • Micro Timer 1 load register - Modulus down counter value of the Micro Timer 1.
        This item modifies the PITMTLD1 register.
      • Frequency of the Micro Timer 0 - Frequency. This item is influenced by the following properties: Micro Timer 0 Load register.
      • Frequency of the Micro Timer 1 - Frequency. This item is influenced by the following properties: Micro Timer 1 Load register.
    • Stop in wait mode - This property is used for power conservation while in wait mode. If set to 'yes' the PIT clock generation stops and freezes the PIT module in the wait mode.
    • Stop in freeze mode - This property is used for power conservation while in wait mode. If set to 'yes' the PIT clock generation stops and freezes the PIT module in the wait mode.
    • Settings only if peripheral contains the PIT trigger.
      • Input Trigger settings - Input Trigger settings
        • Interrupt - Input Trigger interrupt settings.
          • Input Trigger interrupt - Enable/disable input Trigger interrupt.
            This item modifies the PITTRIGIE bit in the PITTRIGCTL register.
          • Interrupt - Interrupt vector (for information only).
          • Priority - Priority of input Trigger interrupt.
          • ISR name - Name of the interrupt service routine (ISR) invoked by this interrupt vector.
            Note: The routine must handle all interrupt flags correctly. The routine must be implemented in the user code, it is not generated by Processor Expert. Find external prototypes of the ISRs in the component module header file.
        • Input edge select - Select one of 3 possible kinds of events on the PIT trigger input.
          This item modifies the PITTRIEDGE register.
        • Input source select - Select one of the four possible PIT input trigger sources.
          This item modifies the PITTRIEDGE register.
  • Channels - Settings of the PIT channels.
    One Item of the list looks like:
    Channel - A PIT timer.
    • Device - Selection of a PIT timer.
    • Enable Timer - Enable/Disable one Timer of a PIT module.
      This item modifies the PCEx bit in the PITCE register.
    • Timer is connected to - Select of the corresponding micro time base.
      This item modifies the PMUXx bits in the PITMUX register.
    • Timer Load register - Modulus down counter value of the Timer x.
      This item modifies the PITLDx register.
    • Frequency of Timer - Frequency. This item is influenced by the following properties: Timer Load register, Micro Timer 0 Load register or Micro Timer 1 Load register.
    • Interrupt - Timer out interrupt settings.
      • Timer out interrupt - Enable/disable Timer out interrupt.
        This item modifies the PITINTEx bit in the PITINTE register.
      • Interrupt - Interrupt vector (for information only).
      • Priority - Priority of timer out interrupt.
      • ISR name - Name of the interrupt service routine (ISR) invoked by this interrupt vector.
        Note: The routine must handle all interrupt flags correctly. The routine must be implemented in the user code, it is not generated by Processor Expert. Find external prototypes of the ISRs in the component module header file.
    • Settings only if peripheral contains the PITCSTP register.
      • Channel stop - If set to 'yes', the PIT channel stops after the next time-out event.
        If set to 'no', the PIT channel restarts after the next time-out event.
        This item modifies the PITCSTPx bit in the PITCSTP register.
    • Settings only if peripheral contains PIT trigger.
      • Trigger - Trigger settings.
        • Combined output trigger - Enable the time-out event of the corresponding channel contributes to the generation of the combined trigger output.
          This item modifies the PITCOTEx bit in the PITTRIGOUT register.
        • Input trigger channel - Enable the start of the corresponding channel is configured to be aligned to the external input trigger event.
          This item modifies the PITTRIGEx bit in the PITTRIGE register.
  • Initialization - Initialization options of the module.
    • PIT periphery - PIT periphery enable.
      This item modifies the PITE bit in the PITCFLMT register.
