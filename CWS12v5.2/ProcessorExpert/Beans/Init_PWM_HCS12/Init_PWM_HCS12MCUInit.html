Component Init_PWM for HCS12
Pulse Width Modulator (PWM)
Device Initialization
(Description of the component parameters.)
  • Device - PWM device (for information only).
  • Settings - Common PWM module settings.
    • Clock settings - Clock/timing settings of the module.
      • Prescaler B select - Prescaler Select for Clock B.
        This item modifies PCKB[2:0] bits in the PWMPRCLK register.

        There are 8 options:
        • 1: This item determine the rate of clock A (B) from bus clock
        • 2: This item determine the rate of clock A (B) from bus clock
        • 4: This item determine the rate of clock A (B) from bus clock
        • 8: This item determine the rate of clock A (B) from bus clock
        • 16: This item determine the rate of clock A (B) from bus clock
        • 32: This item determine the rate of clock A (B) from bus clock
        • 64: This item determine the rate of clock A (B) from bus clock
        • 128: This item determine the rate of clock A (B) from bus clock

      • Clock B frequency - Frequency of clock B (for information only).
        This item is influenced by the following properties: Prescaler B select.
      • Prescaler A select - Prescaler Select for Clock A.
        This item modifies PCKA[2:0] bits in the PWMPRCLK register.

        There are 8 options:
        • 1: This item determine the rate of clock A (B) from bus clock
        • 2: This item determine the rate of clock A (B) from bus clock
        • 4: This item determine the rate of clock A (B) from bus clock
        • 8: This item determine the rate of clock A (B) from bus clock
        • 16: This item determine the rate of clock A (B) from bus clock
        • 32: This item determine the rate of clock A (B) from bus clock
        • 64: This item determine the rate of clock A (B) from bus clock
        • 128: This item determine the rate of clock A (B) from bus clock

      • Clock A frequency - Frequency of Clock A (for information only).
        This item is influenced by the following properties: Prescaler A select.
      • Scale value of clock SB - The programmable scale value used in scaling clock B to generate clock SB.
        This item modifies all bits in the PWMSCLB register.
      • Clock SB frequency - Frequency of clock SB (for information only).
        This item is influenced by the following properties: Prescaler B select, Scale value of clock SB.
      • Scale value of clock SA - The programmable scale value used in scaling clock A to generate clock SA.
        This item modifies all bits in the PWMSCLA register.
      • Clock SA frequency - Frequency of clock SA (for information only).
        This item is influenced by the following properties: Prescaler A select, Scale value of clock SA.
    • Settings only if peripheral has channel 6.
      • Concatenate Channels 6 and 7 - Channels 6 and 7 are concatenated to create one 16-bit PWM channel. Channel 6 becomes the high order byte and channel 7 becomes the low order byte.
        This item modifies CON67 bit in the PWMCTL register.
    • Settings only if peripheral has channel 5.
      • Concatenate Channels 4 and 5 - Channels 4 and 5 are concatenated to create one 16-bit PWM channel. Channel 4 becomes the high order byte and channel 5 becomes the low order byte.
        This item modifies CON45 bit in the PWMCTL register.
    • Settings only if peripheral has channel 3.
      • Concatenate Channels 2 and 3 - Channels 2 and 3 are concatenated to create one 16-bit PWM channel. Channel 2 becomes the high order byte and channel 3 becomes the low order byte.
        This item modifies CON23 bit in the PWMCTL register.
    • Settings only if peripheral has channel 1.
      • Concatenate Channels 0 and 1 - Channels 0 and 1 are concatenated to create one 16-bit PWM channel. Channel 0 becomes the high order byte and channel 1 becomes the low order byte.
        This item modifies CON01 bit in the PWMCTL register.
    • Stop in wait mode - If the property is "yes" it disables the timer module when the MCU is in the wait mode.
      This item modifies PSWAI bit in the PWMCTL register. PSWAI bit also affects pulse accumulator.
    • Stop in freeze mode - If the property is "yes" it disables the timer counter whenever the MCU is in freeze mode.
      This item modifies PFREZ bit in the PWMCTL register. PFREZ bit does not stop the pulse accumulator.
    • Settings only if PWM supports Emergency shutdown feature
      • Emergency shutdown settings - It contains all settings of Emergency shutdown feature (except interrupt).
        • Emergency shutdown feature - Enables Emergency shutdown feature of PWM periphery.
          This item modifies PWM7ENA or PWM5ENA bit in the PWMSDN register.
          The following items are available only if the group is enabled (the value is "Enabled"):
          • Input pin - Input pin used for Emergency shutdown feature.
          • Input pull resistor - Setting of the pull resistor for emergency shutdown pin.
            This item modifies the respective bit in the port input pull-up enable register and eventually in the polarity register.
        • Setting not supported only for Q family derivatives.
          • Input active level - This property determines the active level of the PWM7 channel.
            This item modifies PWM7IN or PWM5IN bit in the PWMSDN register.
          • Shutdown output level - If active level, as defined by the previous property, gets asserted all enabled PWM channels are immediately driven to the level defined by this property.
            This item modifies PWMLVL bit in the PWMSDN register.
  • Channels - Settings of the PWM channels
    One Item of the list looks like:
    Channel0 - PWM channel settings.
    • Counter - Selection of the channel device.
    • PWM enabled - PWM channel enabled. The pulse modulated signal becomes available at the respective PWM output pin when its clock source begins its next cycle.
      This item modifies the PWMEx bit in the PWME register.
    • Polarity - Pulse Width Channel Polarity.
      This item modifies the PPOLx bit in the PWMPOL register.

      There are 2 options:
      • High to low: PWM channel output is high at the beginning of the period, then goes low when the duty count is reached.
      • Low to high: PWM channel output is low at the beginning of the period, then goes high when the duty count is reached.

    • Clock Select - Clock Select. The property sets selected clock source as input to the respective channel counter.
      This item modifies the PCLKx bit in the PWMCLK register. In case PWMCLKAB register is supported, this property modifies appropriate bit(s) of this register too.

      There are 4 options:
      • Clock A: Rate of clock A is determined item "Prescaler A select".
      • Clock SA: Clock SA is generated by taking clock A .
      • Clock B: Rate of clock B is determined item "Prescaler B select".
      • Clock SB: Clock SB is generated by taking clock B.

    • Output Mode - Center Aligned/Left Aligned Output Mode on the respective channel.
      This item modifies the CAEx bit in the PWMCAE register.

      There are 2 options:
      • Center Aligned: CAEx = 1 - Channel operates in Center Aligned Output Mode.
      • Left Aligned: CAEx = 0 - Channel operates in Left Aligned Output Mode.

    • Channel Period - Channel period.
      This item modifies all bits in PWMPERx register.
    • Period time - Period length (for information only).
      This item is influenced by the following properties: Clock Select, Output Mode, Channel period, Prescaler A (or B), Scale value of clock SA (or SB).
    • Channel Duty - Channel duty.
      This item modifies all bits in PWMDTYx register.
    • Duty time - Duty time (for information only). This item is influenced by the following properties: Polarity, Channel duty, Channel period.
    • Pin allocation - The property enables/disables allocation of the PWM channel pin. Allocated pin is exclusively used by this peripheral and cannot be used by any other device.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Pin - Pin used for output of the generated PWM signal.
        Derivatives listed below can route the output pin to the other IO port. This routing feature is intended to support PWM channel availability in low pin count packages.
        Note that if routing is used on high pin count packages the PWM output is mapped to the both IO ports.
        MC9S12C32, MC9S12E128, MC9S12E256, MC9S12GC16 and MC9S12Q128 derivatives only: This item modifies the MODRRx bit in the MODRR register.
      • Pull resistor - Setting of the pull resistor for a PWM channel pin.
        This item modifies the respective bit in the port input pull-up enable register and eventually in the polarity register.
      • Pin open drain - Setting of the open drain option for a PWM channel pin.
        This item modifies the respective bit in the port wired-or mode register.
  • Settings only if PWM supports Emergency shutdown feature
    • Settings not supported for Q family derivatives only.
      • Interrupts - Interrupts settings of the module.
        • Emergency interrupt - Emergency interrupt settings.
          • Emergency interrupt - Emergency interrupt enabled/disabled.
            This item modifies PWMIE bit in the PWMSDN register.
          • Interrupt - Interrupt vector (for information only).
          • Priority - Priority of the emergency interrupt.
          • ISR Name - Name of the interrupt service routine (ISR) invoked by this interrupt vector.
            Note: The routine must handle all interrupt flags correctly. The routine must be implemented in the user code, it is not generated by Processor Expert. Find external prototypes of the ISRs in the component module header file.
  • Initialization - Initialization options of the module.
    • Settings only if PWM supports Emergency shutdown feature
      • Setting not supported only for Q family derivatives.
        • PWM Restart - PWM Restart.
          This item modifies PWMRSTRT bit in the PWMSDN register.
