Component Init_LCD for HCS12X
Liquid Crystal Display (LCD)
Component Level: Peripheral Initialization
Category: CPU Internal Peripherals-Peripheral Initialization
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • Device - Selection of the LCD device module
  • Settings - Common LCD module settings
    • Clock settings - Clock/timing settings of the module
      • LCD clock prescaler - Waveform base clock prescaler of the LCD driver.
        This item modifies LCLK[2:0] bits in the LCDCR0 register
      • LCD module clock - LCD module clock (for information only).
        This item is influenced by the following properties: Clock source select, Extra prescaler by 16, Clock source divider.
      • Display mode - This item selects the display mode of the LCD module driver.
        This item modifies DUTY[1:0] and BIAS bits in the LCDCR0 register
      • Frame frequency - Frame frequency value (for information only).
        This item is influenced by the following properties: Clock source select, Base clock prescaler, Extra prescaler by 16, Clock source divider,Display mode.
    • Stop in Wait mode - Allows or disables the LCD driver whenever the MCU goes into wait mode.
      This item modifies the LCDSWAI bit in the LCDCR1 register.
    • Settings only if LCD supports run in pseudo-stop mode.
      • Run in Pseudo Stop mode - Allows or disables the LCD driver whenever the MCU goes into Pseudo Stop mode.
        This item modifies the LCDRPSTP bit in the LCDCR1 register.
  • Pins - Module pins/signals settings
    • Frontplane pins - Frontplane pins which are allocated by this LCD driver.
      One Item of the list looks like:
      Frontplane pin 0 - One frontplane pin.
      • Frontplane pin - This item enables a Frontplane pin.
      • Frontplane pin signal - Signal name of the pin.
    • Backplane pin 0 - First Backplane pin.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Backplane pin - This item enables a Backplane pin.
      • Backplane pin signal - Signal name of the pin.
    • Backplane pin 1 - Second Backplane pin.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Backplane pin - This item enables a Backplane pin.
      • Backplane pin signal - Signal name of the pin.
    • Backplane pin 2 - Third Backplane pin.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Backplane pin - This item enables a Backplane pin.
      • Backplane pin signal - Signal name of the pin.
    • Backplane pin 3 - Fourth Backplane pin.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Backplane pin - This item enables a Backplane pin.
      • Backplane pin signal - Signal name of the pin.
