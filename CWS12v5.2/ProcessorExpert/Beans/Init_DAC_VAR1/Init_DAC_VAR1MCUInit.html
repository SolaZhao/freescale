Component Init_DAC for VAR1
Digital to analog converter with internal operational amplifier (DAC), currently supported for HCS12 family
Device Initialization
(Description of the component parameters.)
  • Device - Selection of the DAC device module.
  • Settings - Common DAC module settings
    • Device mode - Selects the mode of DAC device and internal operational amplifier (OpAmp).

      There are 5 options:
      • Off: Device disabled.
      • OpAmp: Operational amplifier enabled only.
      • Unbuffered DAC: Unbuffered DAC enabled only.
      • Unbuffered DAC and OpAmp: Unbuffered DAC and independent operational amplifier enabled.
      • Buffered DAC: Buffered DAC enabled only.

    • Voltage range - Voltage range of the DAC output.

      There are 2 options:
      • Full: DAC resistor network operates with the full voltage range.
      • Reduced: DAC resistor network operates with the reduced voltage range.

    • Output voltage value - Defines value of the Output voltage register.
    • Drive select - Selects the drive strength for OpAmp output and buffered DAC output.

      There are 2 options:
      • High: High output drive for low resistive loads.
      • Low: Low output drive for high resistive loads.

  • Pins/Signals - DAC and OpAmp pins/signals settings
    • DAC - DAC related signals/pins
      • Unbuffered DAC output - Unbuffered DAC output. This pin is used in "Unbuffered DAC" and "Unbuffered DAC and OpAmp" mode chosen by Device mode property.
        The following items are available only if the group is enabled (the value is "Enabled"):
        • Pin - Unbuffered DAC output pin.
    • OpAmp - OpAmp related signals/pins
      • Inverted input - OpAmp inverted input. This pin is used in "OpAmp" and "Unbuffered DAC and OpAmp" mode chosen by Device mode property.
        The following items are available only if the group is enabled (the value is "Enabled"):
        • Pin - OpAmp inverted input pin.
      • Non-inverted input - OpAmp non-inverted input. This pin is used in "OpAmp" and "Unbuffered DAC and OpAmp" mode chosen by Device mode property.
        The following items are available only if the group is enabled (the value is "Enabled"):
        • Pin - OpAmp non-inverted input pin.
      • OpAmp/Buffered DAC output - OpAmp/Buffered DAC output settings. This pin is used in "OpAmp", "Unbuffered DAC and OpAmp" and "Buffered DAC" mode chosen by Device mode property.
        The following items are available only if the group is enabled (the value is "Enabled"):
        • Pin - OpAmp/Buffered DAC output pin.
