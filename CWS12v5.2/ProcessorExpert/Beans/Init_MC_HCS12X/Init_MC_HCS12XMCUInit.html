Component Init_MC for HCS12X
PWM motor controller (MC)
Device Initialization
(Description of the component parameters.)
  • Device - Selection of the MC device module
  • Settings - Common MC module settings
    • Clock settings - Clock/timing settings of the module
      • Prescaler - Set value of the prescaler.
        This item modifies MCPRE[1:0] bits in the MCCTL0 register.
      • Period counter - Set value of motor controller timer counter clocks a PWM period lasts.
        This item modifies P[10:0] bits in the MCPER register
      • Dither Feature Enable - Dither mode is to increase the minimum length of output pulses without decreasing the PWM resolution, in order to limit the pulse distortion introduced by the slew rate control of the outputs.
        This item modifies DITH bit in the MCCTL0 register.
      • Period time - Period length counter (for information only).
        This item is influenced by the following properties: Prescaler, Period counter and Dither Feature Enable
    • Recirculation mode - This item determines the flow of the recirculation current of the load.
      This item has no influence when 'Output mode' property is set to 'Half-H bridge' mode.
      This item modifies RECIRC bit in the MCCTL1 register in 'Half-H bridge' mode.
      This item modifies RECIRC bit in the MCCTL1 and S bit in appropriate MCDCx register in '(Dual) Full-H bridge' mode.
    • Fast mode - This item selects the resolution of PWM.
      This item modifies FAST bit in the MCCTL0 register.
    • Stop in wait mode - If the property "yes" it disables the timer module when MCU is in the wait mode.
      This item modifies MCSWAI bit in the MCCTL0 register.
  • Motors - This item increases or decreases number of used motors.
    One Item of the list looks like:
    Motor0 - Motor control group.
    • PWM channel pair - Selection of the PWM channel pair.
    • Coil 0 - Enable/disable PWM channel pair x alignment settings.
      This item modifies MCAM[1:0] bits in the MCCCx register.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Channel mode - The property determines PWM alignment mode.
        This item modifies MCAM[1:0] bits in the MCCCx register.
      • Output mode - This item selects output mode.
        This item modifies MCOM[1:0] bits in the MCCCx register.
      • PWM conected - This item is used to define which output will drive the PWM signal.
        This item modifies S bit in the MCDCx register.
        The value S bit is depending on setting item Recirculation mode.
      • Channel delay - This item selects individually delayed by a programmable number of PWM timer counter clocks.
        This item modifies CD[1:0] bits in the MCCCx register.
      • Channel duty - This value determine the number of motor controller timer counter clocks the corresponding output is driven low or is driven high. Values greater than or equal to the contents of the PWM period will generate a static level output.
        This item modifies D[10:0] bits in the MCDCx register in 11-bit resolution mode.
        This item modifies D[8:2] bits in the MCDCx_FAST register in 7-bit resolution mode.
      • Motor channel period - Period length PWM channel_x (for information only).
        This item is influenced by the following properties: Prescaler, Period counter, Dither Feature and Channel mode
      • Duty - Duty PWM channel_x (for information only).
        This item is influenced by the following properties: Period counter and Channel duty
      • Active state of the PWM signal - Active state of the PWM signal (for information only). This item specifies that the active state of PWM signal is low or high.
      • Plus Pin - Plus pin group.
        This item is 'Disabled' when 'Output mode' property is set to 'Half H-bridge, PWM on MnCxM' mode otherwise it is 'Enabled'.
        The following items are available only if the group is enabled (the value is "Enabled"):
        • Pin - Select pin name.
      • Minus Pin - Minus pin group.
        This item is 'Disabled' when 'Output mode' property is set to 'Half H-bridge, PWM on MnCxP' mode otherwise it is 'Enabled'.
        The following items are available only if the group is enabled (the value is "Enabled"):
        • Pin - Select pin name.
    • Coil 1 - Enable/disable PWM channel pair x+1 alignment settings.
      This item modifies MCAM[1:0] bits in the MCCCx register.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Channel mode - The property determines PWM alignment mode.
        This item modifies MCAM[1:0] bits in the MCCCx register.
      • Output mode - This item selects output mode.
        This item modifies MCOM[1:0] bits in the MCCCx register.
      • PWM conected - This item is used to define which output will drive the PWM signal.
        This item modifies S bit in the MCDCx register.
        The value S bit is depending on setting item Recirculation mode.
      • Channel delay - This item selects individually delayed by a programmable number of PWM timer counter clocks.
        This item modifies CD[1:0] bits in the MCCCx register.
      • Channel duty - This value determine the number of motor controller timer counter clocks the corresponding output is driven low or is driven high. Values greater than or equal to the contents of the PWM period will generate a static level output.
        This item modifies D[10:0] bits in the MCDCx register in 11-bit resolution mode.
        This item modifies D[8:2] bits in the MCDCx_FAST register in 7-bit resolution mode.
      • Motor channel period - Period length PWM channel_x (for information only).
        This item is influenced by the following properties: Prescaler, Period counter, Dither Feature and Channel mode
      • Duty - Duty PWM channel_x (for information only).
        This item is influenced by the following properties: Period counter and Channel duty
      • Active state of the PWM signal - Active state of the PWM signal (for information only). This item specifies that the active state of PWM signal is low or high.
      • Plus Pin - Plus pin group.
        This item is 'Disabled' when 'Output mode' property is set to 'Half H-bridge, PWM on MnCxM' mode otherwise it is 'Enabled'.
        The following items are available only if the group is enabled (the value is "Enabled"):
        • Pin - Select pin name.
      • Minus Pin - Minus pin group.
        This item is 'Disabled' when 'Output mode' property is set to 'Half H-bridge, PWM on MnCxP' mode otherwise it is 'Enabled'.
        The following items are available only if the group is enabled (the value is "Enabled"):
        • Pin - Select pin name.
  • Interrupts - Interrupts setting of the module.
    • MC interrupt - MC interrupt settings
      • Interrupt - Interrupt vector (for information only).
      • MC Interrupt overflow - Enabled/disable MC overflow interrupt.
        This item modifies the MCTOIE bit in MCCTL1 register.
      • Priority interrupt - Priority of the emergency interrupt.
      • ISR name - Name of the interrupt service routine (ISR) invoked by this interrupt vector. The user code of the routine should handle all respective interrupt flags correctly. The routine must be implemented in the user code, it is not generated by Processor Expert. Find external prototypes of the ISRs in the component module header file.
