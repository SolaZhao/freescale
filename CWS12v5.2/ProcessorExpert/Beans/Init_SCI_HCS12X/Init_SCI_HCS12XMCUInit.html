Component Init_SCI for HCS12X
Serial Communications Interface (SCI)
Device Initialization
(Description of the component parameters.)
  • Device - Selection of the SCI communication device module.
  • Settings - Common SCI module settings.
    • Clock settings - Clock/timing settings of the module.
      • Baud rate divisor - Baud rate divisor value used for setting SCI communication speed.
        This item modifies bits form 0 to 12 in the SCIBDH and SCIBDL registers.
      • Baud rate - Calculated SCI communication speed (for information only).
        • Derivatives with infrared option:
          The value is calculated using the following formula:
          • when infrared module is disabled
            baud rate = SCI bus clock / (16 x SBR[12:0])
          • when infrared module is enabled
            baud rate = SCI bus clock / (32 x SBR[12:1])
          This item is influenced by the following properties: Baud rate divisor and Infrared module.
        • Derivatives without infrared option:
          The value is calculated using the following formula: baud rate = SCI bus clock / (16 x SBR[12:0])
          This item is influenced by the following properties: Baud rate divisor.
    • Loop mode - Enable/Disable loop mode of SCI module. Both the transmitter and the receiver must be enabled to use the loop function.
      This item modifies the LOOPS bit in the SCICR1 register.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Function - Select source for SCI receiver when loop mode is enabled.
        This item modifies the RSRC bit in the SCICR1 register.
        The following items are available only if the group is enabled (the value is "Single-wire mode"):
        • TxD direction in Single-Wire mode - This bit determines whether the TXD pin is going to be used as an input or output, in the Single-Wire mode.
          This item modifies the TXDIR bit in the SCISR2 register.
    • Break character length - Item determines whether the transmit break character is 10 or 11 bit respectively 13 or 14 bits long.
      This item modifies the BRK13 bit in the SCISR2 register.
    • Receiver wake up mode - Receiver wake up mode can be set to standby state or normal operation mode.
      This item modifies the RWU bit in the SCICR2 register.
    • Wake up - Select condition for wake up of the SCI device.
      This item modifies the WAKE bit in the SCICR1 register.
    • Idle Line counting - Item determines when the receiver starts counting logic 1s as idle character bits.
      This item modifies the ILT bit in the SCICR1 register.
    • Stop in Wait mode - Enable/Disable running of SCI communication in Wait mode.
      This item modifies the SCISWAI bit in the SCICR1 register.
    • Parity - Select the parity of the data bits.
      This item modifies the PT and PE bits in the SCICR1 register.
    • Data format - Select communication data format.
      This item modifies the M bit in the SCICR1 register.
    • Settings supported for Freescale HCS12 G, HY, P, VR, E, NE, UF, HN64, HZ256, HZ64 and HCS12X derivatives only.
      • Infrared module - This item enables/disables the infrared modulation/demodulation submodule.
        This item modifies the IREN bit in the SCIBDH register.
      • Transmitter Narrow Pulse - This item selects whether the SCI transmits a 1/16, 3/16, or 1/32 narrow pulse.
        This item modifies the TNP0 and TNP1 bits in the SCIBDH register.
    • Settings supported for Freescale HCS12 G, HY, P, VR, UF, E256, HN64, HZ256, HZ64 and HCS12X derivatives only.
      • Transmit polarity - This property selects polarity of the transmitted data.
        This item modifies the TXPOL bit in the SCISR2 register.
      • Receive polarity - This property selects polarity of the received data.
        This item modifies the RXPOL bit in the SCISR2 register.
    • Settings supported for Freescale HCS12 G, HY, P and VR and HCS12X derivatives only.
      • Break detect - This property enables the break detect circuitry.
        This item modifies the BKDFE bit in the SCIACR2 register.
      • Bit error mode - This property determines the functionality of the bit error detect feature.
        This item modifies the BERRM[1:0] bits in the SCIACR2 register.

        There are 3 options:
        • Disabled: Bit error detect circuit is disabled
        • During the 9th time tick: Receive input sampling occurs during the 9th time tick of a transmitted bit
        • During the 13th time tick: Receive input sampling occurs during the 13th time tick of a transmitted bit

  • Pins - Module pins/signals settings.
    • RxD pin - RxD pin properties.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • RxD pin - Receiver input pin used for communication.
      • RxD pull resistor - Setting of the pull resistor for the RxD pin.
        This item modifies the respective bit in the port input pull-up enable register and eventually in the polarity select register.
    • TxD pin - TxD pin properties.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • TxD pin - Transmitter output pin used for communication.
      • TxD pull resistor - Setting of the pull resistor for the TxD pin.
        This item modifies the respective bit in the port input pull-up enable register and eventually in the polarity select register.
      • TxD pin open drain - Setting of the open drain option for the TxD pin.
        This item modifies the respective bit in the port wired-or mode register.
  • Interrupts - Interrupts settings of the module.
    • SCI interrupt - SCI interrupt settings.
      • Interrupt - Interrupt vector (for information only).
      • Receiver full interrupt - Enable/Disable SCI Receiver Full interrupt.
        This item modifies the RIE bit in the SCICR2 register.
      • Transmitter empty interrupt - Enable/Disable SCI Transmitter empty interrupt.
        This item modifies the TIE bit in the SCICR2 register.
      • Transmission complete interrupt - Enable/Disable SCI Transmission complete interrupt.
        This item modifies the TCIE bit in the SCICR2 register.
      • Idle Line interrupt - Enable/Disable SCI Idle Line interrupt.
        This item modifies the ILIE bit in the SCICR2 register.
      • Settings supported for Freescale HCS12 G, HY, P and VR and HCS12X derivatives only.
        • Input active edge interrurpt - Enable/Disable SCI Receive Input Active Edge interrupt.
          This item modifies the RXEDGIE bit in the SCIACR1 register.
        • Bit error interrupt - Enable/Disable SCI Bit Error interrupt.
          This item modifies the BERRIE bit in the SCIACR1 register.
        • Break detect interrupt - Enable/Disable SCI Break Detect interrupt.
          This item modifies the BKDIE bit in the SCIACR1 register.
      • Interrupt priority - Priority of SCI interrupt.
      • ISR name - Name of the interrupt service routine (ISR) invoked by this interrupt vector.
        Note: The routine must handle all interrupt flags correctly. The routine must be implemented in the user code, it is not generated by Processor Expert. Find external prototypes of the ISRs in the component module header file.
  • Initialization - Initialization options of the module.
    • Transmitter - SCI Transmitter enable.
      This item modifies the TE bit in the SCICR2 register.
    • Receiver - SCI receiver enable.
      This item modifies the RE bit in the SCICR2 register.
