Component Init_SPI for HCS12
Serial Peripheral Interface (SPI)
Component Level: Peripheral Initialization
Category: CPU Internal Peripherals-Peripheral Initialization
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • Device - Selection of the SPI communication device module.
  • Settings - Common SPI module settings.
    • Clock settings - Clock/timing settings of the module.
      • Value of Preselection - SPI Baud Rate preselection value used for setting SPI communication frequency.
        This item modifies the SPPR0, SPPR1 and SPPR2 bits in the SPIBR register.
      • Value of Selection - SPI Baud Rate selection value used for setting SPI communication frequency.
        This item modifies the SPR0,SPR1 and SPR2 bits in the SPIBR register.
      • Frequency - Communication speed (for information only]. This item is influenced by the following properties: Value of Preselection, Value of Selection.
    • Mode Select - SPI Master/Slave Mode Select.
      This item modifies the MSTR bit in the SPICR1 register.

      There are 2 options:
      • Master: Master mode select.
      • Slave: Slave mode select

    • Clock Polarity - This item selects an inverted or non-inverted SPI clock. To transmit data between SPI modules, the SPI modules must have identical Clock polarity values.
      This item modifies the CPOL bit in the SPICR1 register.

      There are 2 options:
      • active-high: Active-high clocks selected. In idle state SCK is low.
      • active-low: Active-low clocks selected. In idle state SCK is high.

    • Clock Phase - This item is used to select the SCK serial clock format.
      This item modifies the CPHA bit in the SPICR1 register.

      There are 2 options:
      • First edge: Sampling of data occurs at odd edges (1,3,5,...,15) of the SCK clock.
      • Second edge: Sampling of data occurs at even edges (2,4,6,...,16) of the SCK clock.

    • Data shift order - SPI LSB-First Enable. This item does not affect the position of the MSB and LSB in the data register. Reads and writes of the data register always have the msb in bit 7.
      This item modifies the LSBFE bit in the SPICR1 register.
    • Bidirectional Mode - Bidirectional Pin Configurations.
      This item modifies the SPC0 bit in the SPICR2 register.
    • Output enable in Bidirect. - This item along with the Mode Select item is used to enable the output buffer when the SPI is configured in bidirectional mode.
      This item modifies the BIDIROE bit in the SPICR2 register.
    • SS pin function - Select the function of the Slave Select pin in master mode. In slave mode the SS pin always acts as the low-true slave select input.
      This item modifies the SSOE bit in the SPICR1 register and the MODFEN bit in the SPICR2 register.

      There are 3 options:
      • General Purpose I/O: Slave Select not used by SPI.
      • Slave Select input for mode fault: The pin acts as the mode fault input.
      • Automatic Slave Select output: The pin acts as the slave select output.

    • Stop in Wait Mode - SPI Stop in Wait Mode.
      This item modifies the SPISWAI bit in the SPICR2 register.
    • Settings only if SPI support 16 bit width of data.
      • Data Width - Number of information bits.
        This item modifies the XFRW bit in the SPICR2 register.
  • Pins - Module pins/signals settings.
    • SCK pin - SCK (Serial Clock) pin properties.
      • SCK pin - Clock signal pin.
      • SCK pin signal - Signal name of SCK pin.
      • SCK pull resistor - Setting of the pull resistor for the SCK pin.
        This item modifies the respective bit in the port input pull-up enable register and eventually in the polarity select register.
      • SCK pin open drain - Setting of the open drain option for the SCK pin.
        This item modifies the respective bit in the port wired-or mode register.
    • MISO pin allocation - Enable/disable allocation of the MISO (Master In/Slave Out) pin. Allocated pin is exclusively used by this peripheral and cannot be used by any other device.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • MISO pin - Master in, slave out.
      • MISO pin signal - Signal name of MISO pin.
      • MISO pull resistor - Setting of the pull resistor for the MISO pin.
        This item modifies the respective bit in the port input pull-up enable register and eventually in the polarity select register.
      • MISO pin open drain - Setting of the open drain option for the MSIO pin.
        This item modifies the respective bit in the port wired-or mode register.
    • MOSI pin allocation - Enable/disable allocation of the MOSI (Master Out/Slave In) pin. Allocated pin is exclusively used by this peripheral and cannot be used by any other device.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • MOSI pin - Master out, Slave in.
      • MOSI pin signal - Signal name of MOSI pin.
      • MOSI pull resistor - Setting of the pull resistor for the MOSI pin.
        This item modifies the respective bit in the port input pull-up enable register and eventually in the polarity select register.
      • MOSI pin open drain - Setting of the open drain option for the MOSI pin.
        This item modifies the respective bit in the port wired-or mode register.
    • SS pin allocation - Enable/disable allocation of the SS (Slave Select) pin. Allocated pin is exclusively used by this peripheral and cannot be used by any other device.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • SS pin - Select signal pin.
      • SS pin signal - Signal name of SS pin.
      • SS pull resistor - Setting of the pull resistor for the SS pin.
        This item modifies the respective bit in the port input pull-up enable register and eventually in the polarity select register.
      • SS pin open drain - Setting of the open drain option for the SS pin.
        This item modifies the respective bit in the port wired-or mode register.
  • Interrupts - Interrupts settings of the module.
    • SPI interrupt - SPI interrupt settings.
      • Interrupt - Interrupt vector (for information only).
      • SPI receive and fault interrupt - Enable/disable SPI receive and mode fault interrupt.
        This item modifies the SPIE bit in the SPICR1 register.
      • SPI transmit interrupt - Enable/disable SPI transmit interrupt.
        This item modifies the SPTIE bit in the SPICR1 register.
      • Interrupt Priority - Priority of SPI interrupt.
      • ISR name - Name of the interrupt service routine (ISR) invoked by this interrupt vector.
        Note: The routine must handle all interrupt flags correctly. The routine must be implemented in the user code, it is not generated by Processor Expert. Find external prototypes of the ISRs in the component module header file.
  • Initialization - Initialization options of the module.
    • Call Init method - The Init method of the component may be called after power-on or reset (in initialization code) by PE.
      yes - The Init method is called by PE.
      no - The Init method is not called by PE, it should be called in user's code.
    • Enable SPI system - Enable SPI system .
      This item modifies the SPE bit in the SPICR1 register.
