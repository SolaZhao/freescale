Component FreescaleAnalogComp
Analog comparator on Freescale derivates
Component Level: Low
Category: CPU Internal Peripherals-Measurement
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • Analog comparator - Analog comparator device
  • Interrupt service/event - The component may or may not be implemented using interrupts. If this property is set to "Enabled", the component functionality depends on the interrupt service and will not operate if the CPU interrupts are disabled. For details please refer to chapter "interrupt service in the component's generated code".
    The following items are available only if the group is enabled (the value is "Enabled"):
    • Interrupt - Interrupt vector invoked from the comparator (for information only).
    • Interrupt priority - Priority of the interrupt associated with the comparator.
  • Positive input - Comparator's positive input properties.
    • Pin - Positive analog comparator input pin.
    • Signal - Signal name of the positive comparator input pin.
  • Negative input - Comparator's negative input properties.
    • Pin - Negative analog comparator input pin.
    • Signal - Signal name of the negative comparator input pin
  • Comparator output - Comparator's output properties.
    The following items are available only if the group is enabled (the value is "Enabled"):
    • Pin - Analog comparator output pin.
    • Signal - Signal name of the analog comparator output pin
  • Analog comp. mode - Initial analog comparator mode (edge of analog comparator output to generate interrupt) .

    There are 3 options:
    • falling edge: Interrupt will be generated on comparator output falling edge.
    • rising edge: Interrupt will be generated on comparator output rising edge.
    • both edges: Interrupt will be generated on any edge on comparator output.

  • Initialization - Initial settings (after power-on or reset).
    • Enabled in init. code - The component is enabled after power-on or reset (in initialization code).
    • Event enabled in init. - Event enabled in initialization.
  • CPU clock/speed selection - Settings for the CPU speed modes: if the component is supported or not
    For details about speed modes please refer to page Speed Modes Support.
    • High speed mode - The component is enabled/disabled in high speed mode.
    • Low speed mode - The component is enabled/disabled in low speed mode.
    • Slow speed mode - The component is enabled/disabled in slow speed mode.
