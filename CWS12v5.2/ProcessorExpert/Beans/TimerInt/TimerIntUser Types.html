Component constants:
(Constants defined by this component for usage in the driver interface. They are defined in the generated code.)
  • ComponentName_PM_[Timing] - These constants are used to specify several distinctive modes of timing defined in the Interrupt period property, if the Runtime settings is set to "from list of values". These constants may be used as a parameter for SetPeriodMode method, value is an index of selected user mode. The [Timing] constant name denotes a value of period as it is specified by the user, value without spaces and "." is replaced by "_", e.g. for example 16_384ms for 16.384 ms.
