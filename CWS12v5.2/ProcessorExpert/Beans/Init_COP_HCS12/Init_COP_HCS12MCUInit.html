Component Init_COP for HCS12
Computer Operating Properly (COP)
Device Initialization
(Description of the component parameters.)
  • Device - COP device module (for information only).
  • Settings - Common COP module settings.
    • Clock setting - Clock/timing settings of the module.
      • Settings only if "COP" support "Clock select" ("COP" module has "COPOSCSEL" bit)
        • Clock select - This write-once bit selects the clock source of the COP watchdog.
          This item modifies the COPOSCSEL bit in the CPMUCLKS register.

          There are 3 options:
          • Internal clock: Internal (1-MHz) oscillator is source to COP.
          • External clock: External oscillator is source to COP.
          • Autonomous clock: Trimmable internal RC-oscillator (ACLK) is source to COP.

      • Time-out period prescaler - Time-out period prescaler. This item sets the COP device timeout period. This item modifies the CR[2:0] bits in the COP Control register.
      • Time-out period - Time-out period (for information only).
        This item is influenced by the following properties: Time-out period prescaler.
    • Window mode - Enables COP window mode. In this mode the COP must be reset (a write to the ARMCOP register) in the last 25% of selected period. COP reset during the first 75% of the period will reset the MCU. This item modifies the WCOP bit in the COP Control register.
    • COP stops in wait mode - This property determines a behavior of the COP module in the wait mode. If this property is set to 'yes', the COP module stops and initializes the COP dividers whenever the part goes into Wait Mode. If this property is set to 'no', the COP keeps running in Wait Mode.
      This item modifies the COPWAI bit in the CLKSEL register.
    • Settings only if COP support disable ACLK in stop mode
      • COP stops in stop mode - This property determines a behavior of the COP module in the stop mode. If this property is set to 'yes', COP is disabled in stop mode. Hence the COP is static while in stop mode and continues to operate after exit from stop mode. This property has effect only when Autonomous clock is selected in property "Clock select".
        This item modifies the CSAD bit in the CPMUCLKS register.
    • COP enable during pseudo-stop mode - If set to 'yes', COP continues running in pseudo-stop mode. If set to 'no', COP stops running in pseudo-stop mode.
      This item modifies the PCE bit in the PLLCTL register for derivatives without S12CPMU module.
      This item modifies the PCE bit in the CPMUCLKS register for derivatives with S12CPMU module.
  • Settings only if COP support own interrupt vector
    • Interrupts - Reset settings of the module.
      • COP reset - If Enabled, "ISRname" routine is called from COP reset vector. If Disabled, COP reset vector contains the same address as power on reset vector. Note that in case of using COP reset vector "ISRname" routine, the after-reset initializations of MCU are not done in this "ISRname" routine and the user should take care of them himself.
        The following items are available only if the group is enabled (the value is "Enabled"):
        • Interrupt - Reset vector (for information only)
        • ISR name - Name of the entry point routine invoked by this reset vector. Note: End of this routine must contain jump to power on entry point. See generated code of the component for an example.
