Component TimeDate
Time and date
Component Level: High
Category: CPU Internal Peripherals-Timer
Typical Usage:
(Examples of a typical usage of the component in user code. For more information please see the page Component Code Typical Usage.)

Required component name is "TmDt1".

(1)
There is current time and date read from the component in this example: 

 MAIN.C
 
void main(void)
{
  TIMEREC Time;
  DATEREC Date;
  
  TmDt1_GetTime(&Time);
  TmDt1_GetDate(&Date);

}

(2)
There is alarm used in the following example: 

 MAIN.C
 
extern int Alarm;

void main(void)
{
  TIMEREC Time;
  
  TmDt1_GetTime(&Time); /* get current time */
  
  /* set alarm one minute later */
  TmDt1_SetAlarm(Time.Hour, (Time.Min + 1) % 60, Time.Sec, Time.Sec100);
  
  /* wait till the alarm event (Alarm flag is set) */
  while(!Alarm);
}

 EVENT.C

int Alarm = 0;

void TmDt1_OnAlarm(void)
{
  Alarm = 1; /* set Alarm flag in the alarm event handler */
}
