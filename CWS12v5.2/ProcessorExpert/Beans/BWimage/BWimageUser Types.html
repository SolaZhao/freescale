Component BWimage
Black & White image
Component Level: High
Category: SW-Data
Component types:
(User types are derived from basic types and they are designed for usage in the driver interface. They are declared in the generated code.)
  • TIMAGE = struct { - Black&White image structure
    word width;
    Image width in pixels
    word height;
    Image height in pixels
    byte* pixmap;
    Image pixel bitmap
    size_t size;
    Image size in bytes (actual type is implementation specific, see the current PE_Types.h file)
    char* name;
    Image name
    }
  • PIMAGE = TIMAGE* - Pointer to image structure
Component constants:
(Constants defined by this component for usage in the driver interface. They are defined in the generated code.)
  • ComponentName_IMAGE_WIDTH - The constant representing image width in pixels.
  • ComponentName_IMAGE_HEIGHT - The constant representing image height in pixels.
  • ComponentName_IMAGE_SIZE - The constant representing image size in bytes.
  • ComponentName_IMAGE_FILE_NAME - The constant representing image file name.
  • ComponentName_Pixmap[x] - The constant representing image pixel bitmap. For more details please see General Info page.
