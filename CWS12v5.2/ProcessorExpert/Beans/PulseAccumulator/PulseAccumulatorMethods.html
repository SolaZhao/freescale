Component PulseAccumulator
Pulse Accumulator
Component Level: Low
Category: CPU Internal Peripherals-Timer
Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • Enable - Enables the component.

      ANSIC prototype: byte Enable(void)
    • Return value:byte - Error code, possible codes:
      ERR_OK - OK

  • Disable - Disables the component.

      ANSIC prototype: byte Disable(void)
    • Return value:byte - Error code, possible codes:
      ERR_OK - OK

  • EnableEvent - Enables the component events. This method is only available if any event is selected in the component.

      ANSIC prototype: byte EnableEvent(void)
    • Return value:byte - Error code, possible codes:
      ERR_OK - OK

  • DisableEvent - Disables the component events. This method is only available if any event is selected in the component.

      ANSIC prototype: byte DisableEvent(void)
    • Return value:byte - Error code, possible codes:
      ERR_OK - OK

  • ResetCounter - This method resets the pulse accumulator counter (sets to zero).

      ANSIC prototype: void ResetCounter(void)

  • SetCounter - This method sets the pulse accumulator counter.

      ANSIC prototype: void SetCounter(ComponentName_TCounterValue Value)
    • Value:ComponentName_TCounterValue - This value is stored into a counter register. Note: Parameter type depends on the width of the selected counter (may be 8, 16, 32 or 64 bits long). See the method prototype in the generated module.

  • Methods for HCS12(X) derivatives.
    • GetCounterValue - This method returns the content of the pulse accumulator counter.

        ANSIC prototype: ComponentName_TCounterValue GetCounterValue(void)
      • Return value:ComponentName_TCounterValue - Content of the counter register.

    • Methods only if timer supports holding captured value.
      • GetHoldValue - This method returns the content of the pulse accumulator holding register. This method is only available if "Buffer" property is enabled.

          ANSIC prototype: ComponentName_TCounterValue GetHoldValue(void)
        • Return value:ComponentName_TCounterValue - Content of the holding register.

      • Latch - This method forces the content of the pulse accumulator count register to be latched into the associated holding register. It influences also Capture component on the same ECT peripheral (channel 0-3), please refer to CPU datasheet for details. This method is only available if "Mode" property is set to "Latch".

          ANSIC prototype: void Latch(void)


Note: Some methods can be implemented as macros.
