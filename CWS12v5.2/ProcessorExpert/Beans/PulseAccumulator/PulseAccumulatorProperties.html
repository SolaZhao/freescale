Component PulseAccumulator
Pulse Accumulator
Component Level: Low
Category: CPU Internal Peripherals-Timer
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Component name
  • Counter - Counter register of the pulse accumulator.
  • Interrupt service/event - Component uses the interrupt service. If this property is set to "Enabled", the component functionality (e.g. invoking events) depends on the interrupt service and will not operate if the CPU interrupts are disabled. For details please refer to chapter "interrupt service in the component's generated code".
    The following items are available only if the group is enabled (the value is "Enabled"):
    • Settings supported for HCS12(X) derivatives only.
      • Overflow device - Overflow device
      • Overflow interrupt - Interrupt vector invoked from pulse accumulator (for information only).
      • Overflow int. priority - The priority of the interrupt associated with the pulse accumulator.
      • Edge interrupt - Interrupt vector invoked from pulse accumulator (for information only).
      • Edge int. priority - The priority of the interrupt associated with the input device.
  • Settings supported for HCS12(X) derivatives only.
    • Initialize input - This property allows sharing of input pin with Capture component: if it is shared, disable this property. Disable is supported only for ECT timer: PACNT0, PACNT1, PACNT01, PACNT2 and PACNT3.

      If the initialization is disabled without sharing, it is necessary to initialize input manually in the user code.
      The following items are available only if the group is enabled (the value is "Enabled"):

      • Input pin - Input pin.
      • Input pin signal - Signal name of input pin.
      • Pull resistor - Setting of pull resistor.

        There are 6 options:
        • no pull resistor: Input without pull resistor is required
        • pull up: Input with pull-up resistor is required
        • pull down: Input with pull-down resistor is required
        • pull up or no pull: Input with pull-up resistor is preferred, but no-pull is acceptable
        • pull down or no pull: Input with pull-down resistor is preferred, but no-pull is acceptable
        • autoselected pull: There are no special requirements for pull resistors

      • Edge - Type of signal edges that can invoke the OnEdge events. The property is not valid for Gated time mode.

        There are 4 options:
        • rising or falling edge: One of the following edge type will be selected: rising or falling
        • rising edge: Rising edge required
        • falling edge: Falling edge required
        • both edges: Both edges (rising and falling) required

      • Settings only if timer supports holding captured value.
        • Overwrite - The contents of the related capture or holding registers can be overwritten when a new input capture or latch occurs.
    • Mode - Pulse accumulator mode.
      • In the event counter mode, the pulse accumulator counts the number of active edges on the input pin.
      • In gated time accumulation mode, a pulse accumulator counts the internal timer clock gated by the input pin. On ECT timer the gated mode is available only for PACNT23.

      The following items are available only if the group is enabled (the value is "Gated time"):
      • Count level - The select input pin level that enables the clock for the pulse accumulator. Note: property 'Edge' (in 'Initialize input' group) has no meaning for this mode.
      • Pulse period - Period of the pulse accumulator counting (for information only).
  • Initialization - Initial settings (after power-on or reset).
    • Enabled in init. code - The component is enabled after power-on or reset (in initialization code).
    • Events enabled in init. - Events enabled in initialization.
