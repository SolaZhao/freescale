 Embedded Components Categories

  • CPU External Devices - This category contains components that encapsulate devices, which can be externally connected to the MCU.
    Typically, these components inherit internal peripheral component to provide an independant layer between CPU and device.
    • Communication - This category contains components which cover external communication devices or protocols. Most of these components use standard internal peripheral components to provide independent access to the CPU peripherals.
      • SW_I2C - SW emulated I2C using two I/O pins.
      • SWSPI - Software synchronous serial communication
    • Display - This category contains components for supporting displaying devices such as LEDs, VGA modules, LCDs, Terminals or support of external display chips.
      • Term - ANSI Terminal
    • Port I/O - This category contains components for basic external I/O ports operations.
      • Ext16IO - External 16-bit Input/Output
      • Ext32IO - External 32-bit Input/Output
      • Ext8IO - External 8-bit Input/Output
      • ExtBitIO - External 1-bit Input/Output with Direction Register
      • ExtByteIO - External Byte Input/Output (8 bits) with Direction Register
  • CPU Internal Peripherals - This category contains components for on-chip peripherals.
    • Communication - This category contains components, which cover several communication devices or protocols supported by on-chip peripherals. If a communication protocol supports Master / Slave mode of connected nodes, two separated components with appendix Master / Slave may cover corresponding communication device.
      • AsynchroMaster - Asynchronous serial communication - master
      • AsynchroSerial - Asynchronous serial communication
      • AsynchroSlave - Asynchronous serial communication - slave
      • BDLC - BDLC serial communication
      • Byteflight - Byteflight communication interface
      • FreescaleCAN - CAN communication for Freescale implementation
      • FreescaleEMAC - Configuration modes for MC9S12NE64 EMAC start-up
      • FreescaleEthernetAPI - NE64 ethernet API routines
      • InternalI2C - Internal I2C Communication Interface
      • SynchroMaster - Master for synchronous serial communication
      • SynchroSlave - Slave for synchronous serial communication
    • Converter - This category contains components which cover several on-chip conversion functionality, e.g. A/D or D/A convertors.
      • ADC - This category contains components which cover on-chip A/D convertors and related modules.
        • ADC - A/D converter
      • DAC - This category contains components which cover on-chip D/A convertors and related modules.
        • DAC - D/A converter
    • Interrupts - This category contains components which encapsulate external interrupt pins and also interrupt vector table.
      • ExtInt - External interrupt
      • InterruptVector - Interrupt vector handled in user code
    • Measurement - This category contains components which cover several measurements supported by on-chip peripherals.
      • Capture - Timer capture encapsulation
      • FreescaleAnalogComp - Analog comparator on Freescale derivates
    • Memory - This category contains components which encapsulate support of on-chip memory.
      • IntEEPROM - Internal EEPROM
      • IntFLASH - Internal FLASH
    • Peripheral Initialization - This category contains Peripheral Initialization components, which provide low-level hardware approach to initialize the peripherals.
      • Init_ACMP - Peripheral Initialization Component for HCS12 family - Analog comparator (ACMP)
      • Init_ADC - Peripheral Initialization Component in the following variants:
        -for HCS12X family - Analog-to-Digital Converter (ADC)
        -for HCS12 family - Analog-to-Digital Converter (ADC)
      • Init_API - Peripheral Initialization Component in the following variants:
        -for HCS12X family - Autonomous Periodic Interrupt (API)
        -for HCS12 family - Autonomous Periodic Interrupt (API)
      • Init_BATS - Peripheral Initialization Component multiplatform variant - Supply Voltage Sensor (BATS)., currently supported for HCS12, HCS12Z families
      • Init_BDLC - Peripheral Initialization Component for HCS12 family - Society of Automotive Engineers (SAE) J1850 serial communication network. (BDLC)
      • Init_BYTEFLIGHT - Peripheral Initialization Component for HCS12 family - Byteflight communication interface (BYTEFLIGHT)
      • Init_COP - Peripheral Initialization Component in the following variants:
        -for HCS12X family - Computer Operating Properly (COP)
        -for HCS12 family - Computer Operating Properly (COP)
      • Init_DAC - Peripheral Initialization Component in the following variants:
        -multiplatform variant - Digital to analog converter with internal operational amplifier (DAC), currently supported for HCS12 family
        -for HCS12 family - Digital to Analog Converter (DAC)
      • Init_ECT - Peripheral Initialization Component in the following variants:
        -for HCS12X family - Enhanced Capture Timer (ECT)
        -for HCS12 family - Enhanced Capture Timer (ECT)
      • Init_EEPROM - Peripheral Initialization Component in the following variants:
        -for HCS12X family - EEPROM Memory (EEPROM)
        -for HCS12 family - EEPROM Memory (EEPROM)
      • Init_FLASH - Peripheral Initialization Component in the following variants:
        -for HCS12X family - Flash Memory (FLASH)
        -for HCS12 family - Flash Memory (FLASH)
      • Init_GPIO - Peripheral Initialization Component in the following variants:
        -for HCS12X family - General Purpose Input Output (GPIO)
        -for HCS12 family - General Purpose Input Output (GPIO)
      • Init_HSDRV - Peripheral Initialization Component in the following variants:
        -multiplatform variant - High-Side Drivers (HSDRV), currently supported for HCS12 family
        -for HCS12 family - High-Side Drivers (HSDRV)
      • Init_IIC - Peripheral Initialization Component in the following variants:
        -for HCS12X family - Inter-IC Bus (IIC)
        -for HCS12 family - Inter-IC Bus (IIC)
      • Init_IRQ - Peripheral Initialization Component in the following variants:
        -for HCS12X family - External Interrupt Module (IRQ)
        -for HCS12 family - External Interrupt Module (IRQ)
      • Init_LCD - Peripheral Initialization Component in the following variants:
        -for HCS12X family - Liquid Crystal Display (LCD)
        -for HCS12 family - Liquid Crystal Display (LCD)
      • Init_LINPHY - Peripheral Initialization Component multiplatform variant - LIN Physical Layer (LINPHY), currently supported for HCS12, HCS12Z families
      • Init_LSDRV - Peripheral Initialization Component in the following variants:
        -multiplatform variant - Low-Side Drivers (LSDRV), currently supported for HCS12 family
        -for HCS12 family - Low-Side Drivers (LSDRV)
      • Init_MC - Peripheral Initialization Component in the following variants:
        -for HCS12X family - PWM motor controller (MC)
        -for HCS12 family - PWM motor controller (MC)
      • Init_MSCAN - Peripheral Initialization Component in the following variants:
        -for HCS12X family - Motorola Scalable Controller Area Network (MSCAN)
        -for HCS12 family - Motorola Scalable Controller Area Network (MSCAN)
      • Init_PIT - Peripheral Initialization Component for HCS12X family - Periodic Interrupt Timer (PIT) or Enhanced Programmable Interrupt Timer (EPIT)
      • Init_PMF - Peripheral Initialization Component in the following variants:
        -for HCS12X family - Pulse width Modulator with Fault protection (PMF)
        -for HCS12 family - Pulse width Modulator with Fault protection (PMF)
      • Init_PWM - Peripheral Initialization Component in the following variants:
        -for HCS12X family - Pulse Width Modulator (PWM)
        -for HCS12 family - Pulse Width Modulator (PWM)
      • Init_RTI - Peripheral Initialization Component in the following variants:
        -for HCS12X family - Real Time Interrupt (RTI)
        -for HCS12 family - Real Time Interrupt (RTI)
      • Init_SCI - Peripheral Initialization Component in the following variants:
        -for HCS12X family - Serial Communications Interface (SCI)
        -for HCS12 family - Serial Communications Interface (SCI)
      • Init_SPI - Peripheral Initialization Component in the following variants:
        -for HCS12X family - Serial Peripheral Interface (SPI)
        -for HCS12 family - Serial Peripheral Interface (SPI)
      • Init_SSD - Peripheral Initialization Component for HCS12X family - Stepper Stall Detector (SSD)
      • Init_TIM - Peripheral Initialization Component in the following variants:
        -for HCS12X family - Timer Interface Module (TIM)
        -for HCS12 family - Timer Interface Module (TIM)
      • InterruptVector - Interrupt vector handled in user code
    • Port I/O - This category contains components for basic I/O ports operations.
      • BitIO - General 1-bit Input/Output
      • BitsIO - General Multi-Bits Input/Output (1-8 bits)
      • Byte2IO - General Two-Bytes Input/Output
      • Byte3IO - General Three-Bytes Input/Output
      • Byte4IO - General Four-Bytes Input/Output
      • ByteIO - General Byte Input/Output (8 bits)
      • InputPin - General 1-bit Input
      • WordIO - General Word (16-bit) Input/Output
    • Timer - This category contains components which encapsulate several functionality of timers and related CPU Internal Peripherals.
      • EventCntr16 - Event counter 16-bit
      • EventCntr32 - Event counter 32-bit
      • EventCntr8 - Event counter 8-bit
      • FreeCntr - Time measurement based on counter value (suitable for short time periods)
      • FreeCntr16 - Time measurement counting periodic interrupt ticks in 16-bit SW counter (suitable for longer time periods).
      • FreeCntr32 - Time measurement counting periodic interrupt ticks in 32-bit SW counter (suitable for longer time periods).
      • FreeCntr8 - Time measurement counting periodic interrupt ticks in 8-bit SW counter (suitable for longer time periods).
      • PPG - Programmable pulse generation
      • PulseAccumulator - Pulse Accumulator
      • PWM - Pulse width modulation
      • PWMMC - Pulse width modulation for motor control
      • RTIshared - Real Time Interrupt Shareable
      • TimeDate - Time and date
      • TimerInt - Periodic interrupt
      • TimerOut - Flip-flop output 1:1
      • WatchDog - WatchDog component
  • SW - This category contains components that implement pure software algorithms and are not CPU/peripheral independant.
    Also user-defined components are located here.
    • Communication - This category contains components which cover communication devices or protocols.
      • Term - ANSI Terminal
    • Data - Components for supporting an external data in the project.
      • BWimage - Black & White image
      • COLimage - Color image
      • ExternalFile - External Binary File Converter
      • StringList - String List Converter
    • Operating System - This category contains components which cover several OS configurations.
      • OSEK_OS - Encapsulation of OSEK/VDX compliant OS - HW specific configuration
    • Tutorials And Demonstrations - This category contains components intended for tutorials and demonstrations.
      • BasicProperties - This component is example of basic properties. The component was designed to help developers start creating a new user components quickly.
      • TwoKeys - Component generated by Component Creator
