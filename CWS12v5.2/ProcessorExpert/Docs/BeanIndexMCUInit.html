Index of Peripheral Initialization Components supported for Device Initialization

Installed Peripheral Initialization Components
  • Init_ACMP - Peripheral Initialization Component for HCS12 family - Analog comparator (ACMP)
  • Init_ADC - Peripheral Initialization Component in the following variants:
    -for HCS12X family - Analog-to-Digital Converter (ADC)
    -for HCS12 family - Analog-to-Digital Converter (ADC)
  • Init_API - Peripheral Initialization Component in the following variants:
    -for HCS12X family - Autonomous Periodic Interrupt (API)
    -for HCS12 family - Autonomous Periodic Interrupt (API)
  • Init_BATS - Peripheral Initialization Component multiplatform variant - Supply Voltage Sensor (BATS)., currently supported for HCS12, HCS12Z families
  • Init_BDLC - Peripheral Initialization Component for HCS12 family - Society of Automotive Engineers (SAE) J1850 serial communication network. (BDLC)
  • Init_BYTEFLIGHT - Peripheral Initialization Component for HCS12 family - Byteflight communication interface (BYTEFLIGHT)
  • Init_COP - Peripheral Initialization Component in the following variants:
    -for HCS12X family - Computer Operating Properly (COP)
    -for HCS12 family - Computer Operating Properly (COP)
  • Init_DAC - Peripheral Initialization Component in the following variants:
    -multiplatform variant - Digital to analog converter with internal operational amplifier (DAC), currently supported for HCS12 family
    -for HCS12 family - Digital to Analog Converter (DAC)
  • Init_ECT - Peripheral Initialization Component in the following variants:
    -for HCS12X family - Enhanced Capture Timer (ECT)
    -for HCS12 family - Enhanced Capture Timer (ECT)
  • Init_EEPROM - Peripheral Initialization Component in the following variants:
    -for HCS12X family - EEPROM Memory (EEPROM)
    -for HCS12 family - EEPROM Memory (EEPROM)
  • Init_FLASH - Peripheral Initialization Component in the following variants:
    -for HCS12X family - Flash Memory (FLASH)
    -for HCS12 family - Flash Memory (FLASH)
  • Init_GPIO - Peripheral Initialization Component in the following variants:
    -for HCS12X family - General Purpose Input Output (GPIO)
    -for HCS12 family - General Purpose Input Output (GPIO)
  • Init_HSDRV - Peripheral Initialization Component in the following variants:
    -multiplatform variant - High-Side Drivers (HSDRV), currently supported for HCS12 family
    -for HCS12 family - High-Side Drivers (HSDRV)
  • Init_IIC - Peripheral Initialization Component in the following variants:
    -for HCS12X family - Inter-IC Bus (IIC)
    -for HCS12 family - Inter-IC Bus (IIC)
  • Init_IRQ - Peripheral Initialization Component in the following variants:
    -for HCS12X family - External Interrupt Module (IRQ)
    -for HCS12 family - External Interrupt Module (IRQ)
  • Init_LCD - Peripheral Initialization Component in the following variants:
    -for HCS12X family - Liquid Crystal Display (LCD)
    -for HCS12 family - Liquid Crystal Display (LCD)
  • Init_LINPHY - Peripheral Initialization Component multiplatform variant - LIN Physical Layer (LINPHY), currently supported for HCS12, HCS12Z families
  • Init_LSDRV - Peripheral Initialization Component in the following variants:
    -multiplatform variant - Low-Side Drivers (LSDRV), currently supported for HCS12 family
    -for HCS12 family - Low-Side Drivers (LSDRV)
  • Init_MC - Peripheral Initialization Component in the following variants:
    -for HCS12X family - PWM motor controller (MC)
    -for HCS12 family - PWM motor controller (MC)
  • Init_MSCAN - Peripheral Initialization Component in the following variants:
    -for HCS12X family - Motorola Scalable Controller Area Network (MSCAN)
    -for HCS12 family - Motorola Scalable Controller Area Network (MSCAN)
  • Init_PIT - Peripheral Initialization Component for HCS12X family - Periodic Interrupt Timer (PIT) or Enhanced Programmable Interrupt Timer (EPIT)
  • Init_PMF - Peripheral Initialization Component in the following variants:
    -for HCS12X family - Pulse width Modulator with Fault protection (PMF)
    -for HCS12 family - Pulse width Modulator with Fault protection (PMF)
  • Init_PWM - Peripheral Initialization Component in the following variants:
    -for HCS12X family - Pulse Width Modulator (PWM)
    -for HCS12 family - Pulse Width Modulator (PWM)
  • Init_RTI - Peripheral Initialization Component in the following variants:
    -for HCS12X family - Real Time Interrupt (RTI)
    -for HCS12 family - Real Time Interrupt (RTI)
  • Init_SCI - Peripheral Initialization Component in the following variants:
    -for HCS12X family - Serial Communications Interface (SCI)
    -for HCS12 family - Serial Communications Interface (SCI)
  • Init_SPI - Peripheral Initialization Component in the following variants:
    -for HCS12X family - Serial Peripheral Interface (SPI)
    -for HCS12 family - Serial Peripheral Interface (SPI)
  • Init_SSD - Peripheral Initialization Component for HCS12X family - Stepper Stall Detector (SSD)
  • Init_TIM - Peripheral Initialization Component in the following variants:
    -for HCS12X family - Timer Interface Module (TIM)
    -for HCS12 family - Timer Interface Module (TIM)
